    11. HTML. Комментарии

    Теги комментариев используются для вставки комментариев в исходный код HTML. Вы можете добавлять комментарии к своему HTML-коду, используя следующий синтаксис:

    <!-- Напишите свой комментарий здесь -->

    Обратите внимание, что в открывающем теге есть восклицательный знак (!), но его нет в закрывающем теге.

    Примечание. Комментарии не отображаются браузером, но они могут помочь документировать исходный код HTML.

    С комментариями вы можете размещать уведомления и напоминания в своем HTML:

    <!-- Это комментарий -->
    <p>Это параграф.</p>
    <!-- Не забудьте добавить дополнительную информацию здесь. -->

    Комментарии также хороши для отладки (дебагинга) HTML, потому что вы можете закомментировать HTML-строки кода, по одной, для поиска ошибок:

    <!-- Не показывать это в данный момент
    <img border="0" src="pic_mountain.jpg" alt="Mountain">
    -->

    Условные комментарии

    Вы также можете наткнуться на условные комментарии в HTML:

    <!--[if IE 9]> .... some HTML here .... <![endif]-->

    Условные комментарии определяют некоторые теги HTML, которые должны выполняться только браузером Internet Explorer.

    Проверьте себя с помощью упражнений!
