    HTML5. Руководство

    С помощью HTML вы можете создать свой собственный веб-сайт.

    В этом курсе вы узнаете все о языке разметки веб-страниц HTML.

    HTML легко выучить – и вам это понравится!

    Примеры в каждой главе

    Этот HTML-курс содержит сотни примеров HTML. С помощью онлайн-редактора HTML на сайте https://www.w3schools.com/ вы можете отредактировать HTML-код и нажать кнопку, чтобы просмотреть результат.

    <!DOCTYPE html>
    <html>
    <head>
    <title>Название страницы</title>
    </head>
    <body>
    <h1>Это заголовок</h1>
    <p>Это параграф.</p>
    </body>
    </html>

    Начните изучать HTML сейчас!
