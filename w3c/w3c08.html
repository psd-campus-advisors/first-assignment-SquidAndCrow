    Атрибут style HTML

    Настройка стиля элемента HTML может быть выполнена с помощью атрибута style. Атрибут style имеет следующий синтаксис:

    <tagname style="property:value;">

    где

    tagname - название тега,

    property - свойство CSS,

    value - значение CSS.


    В этом уроке вы узнаете больше о CSS.

    Цвет фона страницы

    Свойство background-color определяет цвет фона для элемента HTML.

    В этом примере задается цвет фона для страницы:

    <body style="background-color:powderblue;">
    <h1>Это заголовок</h1>
    <p>Это параграф.</p>
    </body>

    Цвет фона страницы в этом примере будет голубым.

    Цвет текста

    Свойство color определяет цвет текста для элемента HTML:

    <h1 style="color:blue;">This is a heading</h1> <p style="color:red;">This is a paragraph.</p>

    HTML-шрифты

    Свойство font-family определяет шрифт, который будет использоваться для элемента HTML:

    <h1 style="font-family:verdana;">Это заголовок</h1> <p style="font-family:courier;">Это параграф.</p>

    Размер текста HTML

    Свойство font-size определяет размер текста для элемента HTML:

    <h1 style="font-size:200%;">Это заголовок</h1> <p style="font-size:160%;">Это параграф.</p>

    Выравнивание текста HTML

    Свойство text-align определяет горизонтальное выравнивание текста для элемента HTML:

    <h1 style="text-align:center;">Centered Heading</h1> <p style="text-align:center;">Centered paragraph.</p>

    Проверьте себя с помощью упражнений!

    Создание стиля веб-страниц

    Атрибут, свойства Описание
    style для стилизации HTML-элементов
    background-color для цвета фона
    color для цвета текста
    font-family для названия (семьи) шрифта
    font-size для размера шрифта
    text-align для выравнивания текста
