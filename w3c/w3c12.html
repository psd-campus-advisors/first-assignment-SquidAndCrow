    12. HTML. Цвета

    HTML-цвета указываются с использованием предопределенных имен цветов или значений RGB, HEX, HSL, RGBA, HSLA.

    Имена цветов

    В HTML цвет можно указать с помощью имени цвета:

    <h4 style="background-color:Tomato;">Tomato</h4>
    <h4 style="background-color:Orange;">Orange</h4>
    <h4 style="background-color:DodgerBlue;">DodgerBlue</h4>
    <h4 style="background-color:MediumSeaGreen;">MediumSeaGreen</h4>
    <h4 style="background-color:Gray;">Gray</h4>
    <h4 style="background-color:SlateBlue;">SlateBlue</h4>
    <h4 style="background-color:Violet;">Violet</h4>
    <h4 style="background-color:LightGray;">LightGray</h4>

    Результат:

    Tomato

    Orange

    DodgerBlue

    MediumSeaGreen

    Gray

    SlateBlue

    Violet

    LightGray


    HTML поддерживает 140 стандартных имен цветов.

    Цвет фона (background-color)

    Вы можете установить цвет фона для элементов HTML:

    <h4 style="background-color:DodgerBlue;">Hello World</h1>
    <p style="background-color:Tomato;">Lorem ipsum...</p>

    Результат:

    Hello World

    Lorem ipsum...

    Цвет текста

    Вы можете установить цвет текста:

    <h4 style="color:Tomato;">Hello, World!</h4>
    <p style="color:DodgerBlue;">Слава Україні!</p>
    <p style="color:MediumSeaGreen;">Героям Слава!</p>

    Результат:

    Hello, World!

    Слава Україні!

    Героям Слава!

    Цвет границы

    Вы можете установить цвет границ:

    <h4 style="border:2px solid Tomato;">Hello World</h4>
    <h4 style="border:2px solid DodgerBlue;">Hello World</h4>
    <h4 style="border:2px solid Violet;">Hello World</h4>

    Результат:

    Hello World

    Hello World

    Hello World

    Цветовые значения

    В HTML цвета также могут быть указаны с использованием значений RGB, значений HEX, значений HSL, значений RGBA и значений HSLA:

    То же, что и название цвета «Tomato»:

    <h4 style="background-color:rgb(255, 99, 71);">rgb(255, 99, 71)</h4>
    <h4 style="background-color:#ff6347;">#ff6347</h4>
    <h4 style="background-color:hsl(9, 100%, 64%);">hsl(9, 100%, 64%)</h4>

    Результат:

    rgb(255, 99, 71)

    #ff6347

    hsl(9, 100%, 64%)

    То же, что и название цвета «Tomato», но на 50% прозрачней:

    <h4 style="background-color:rgba(255, 99, 71, 0.5);">255, 99, 71, 0.5</h4>
    <h4 style="background-color:hsla(9, 100%, 64%, 0.5);">9, 100%, 64%, 0.5</h4>

    Результат:

    255, 99, 71, 0.5

    9, 100%, 64%, 0.5

    Значение RGB

    В HTML цвет может быть указан как значение RGB, используя эту формулу:

    rgb(red - красный, green - зеленый, blue - синий)

    Каждый параметр (красный, зеленый и синий) определяет интенсивность цвета от 0 до 255.

    Например, rgb (255, 0, 0) отображается как красный, потому что красный цвет имеет максимальное значение (255), а остальные - 0.

    Чтобы отобразить цвет черный, все цветовые параметры должны быть установлены в 0, например: rgb (0, 0, 0). Чтобы отобразить цвет белый, все параметры цвета должны быть установлены на 255, например: rgb (255, 255, 255).

    Эксперимент смешивания значений RGB ниже:

    <h4 style="background-color:rgb(255, 0, 0);">rgb(255, 0, 0)</h4>
    <h4 style="background-color:rgb(0, 0, 255);">rgb(0, 0, 255)</h4>
    <h4 style="background-color:rgb(60, 179, 113);">rgb(60, 179, 113)</h4>
    <h4 style="background-color:rgb(238, 130, 238);">rgb(238, 130, 238)</h4>
    <h4 style="background-color:rgb(255, 165, 0);">rgb(255, 165, 0)</h4>
    <h4 style="background-color:rgb(106, 90, 205)>rgb(106, 90, 205)</h4>

    Результат:

    rgb(255, 0, 0)

    rgb(0, 0, 255)

    rgb(60, 179, 113)

    rgb(238, 130, 238)

    rgb(255, 165, 0)

    rgb(106, 90, 205)

    Оттенки серого часто определяются с использованием равных значений для всех трех источников света:

    <h4 style="background-color:rgb(0, 0, 0);">rgb(0, 0, 0)</h4>
    <h4 style="background-color:rgb(60, 60, 60);">rgb(60, 60, 60)</h4>
    <h4 style="background-color:rgb(120, 120, 120);">rgb(120, 120, 120)</h4>
    <h4 style="background-color:rgb(180, 180, 180);">rgb(180, 180, 180)</h4>
    <h4 style="background-color:rgb(240, 240, 240);">rgb(240, 240, 240)</h4>
    <h4 style="background-color:rgb(255, 255, 255);">rgb(255, 255, 255)</h4>

    Результат:

    rgb(0, 0, 0)

    rgb(60, 60, 60)

    rgb(120, 120, 120)

    rgb(180, 180, 180)

    rgb(240, 240, 240)

    rgb(255, 255, 255)

    Используя равные значения для красного, зеленого и синего, вы получите разные оттенки серого.

    Значение HEX

    HTML цвет может быть указан с использованием шестнадцатеричного значения в форме:

    #rrggbb

    Где rr (red-красный), gg (green-зеленый) и bb (blue-синий) являются шестнадцатеричными значениями между 00 и ff (такими же, как десятичные 0-255).

    Например, #ff0000 отображается как красный, потому что красный установлен на его самое высокое значение (ff), а остальные установлены на самое низкое значение (00).

    <h4 style="background-color:#ff0000;">#ff0000</h4>
    <h4 style="background-color:#0000ff;">#0000ff</h4>
    <h4 style="background-color:#3cb371;">#3cb371</h4>
    <h4 style="background-color:#ee82ee;">#ee82ee</h4>
    <h4 style="background-color:#ffa500;">#ffa500</h4>
    <h4 style="background-color:#6a5acd;">#6a5acd</h4>

    Результат:

    #ff0000

    #0000ff

    #3cb371

    #ee82ee

    #ffa500

    #6a5acd

    В HTML вы можете указать цвета, используя значения HEX. Большинство сайтов при стилизации использует именно значения HEX.

    Оттенки серого часто определяются с использованием равных значений для всех трех источников света:

    <h4 style="background-color:#000000;">#000000</h4>
    <h4 style="background-color:#3c3c3c;">#3c3c3c</h4>
    <h4 style="background-color:#787878;">#787878</h4>
    <h4 style="background-color:#b4b4b4;">#b4b4b4</h4>
    <h4 style="background-color:#f0f0f0;">#f0f0f0</h4>
    <h4 style="background-color:#ffffff;">#ffffff</h4>

    Результат:

    #000000

    #3c3c3c

    #787878

    #b4b4b4

    #f0f0f0

    #ffffff

    Используя равные значения для красного, зеленого и синего, вы получите разные оттенки серого.

    Значение HSL

    В HTML цвет может быть указан с использованием оттенка, насыщенности и легкости/светлоты (HSL) в форме:

    HSL (hue - оттенок, saturation - насыщенность, lightness - легкость, освещенность)

    Hue - Оттенок

    Оттенок - это градус на цветном круге от 0 до 360. 0 красный, 120 зеленый, а 240 - синий.

    Насыщенность - это процентное значение, 0% означает оттенок серого, а 100% - полный цвет.

    Легкость также составляет процент, 0% - черный, 50% - не светлый или темный, 100% - белый.

    <h4 style="background-color:hsl(0, 100%, 50%);">hsl(0, 100%, 50%)</h4>
    <h4 style="background-color:hsl(240, 100%, 50%);">hsl(240, 100%, 50%)</h4>
    <h4 style="background-color:hsl(147, 50%, 47%);">hsl(147, 50%, 47%)</h4>
    <h4 style="background-color:hsl(300, 76%, 72%);">hsl(300, 76%, 72%)</h4>
    <h4 style="background-color:hsl(39, 100%, 50%);">hsl(39, 100%, 50%)</h4>
    <h4 style="background-color:hsl(248, 53%, 58%);">hsl(248, 53%, 58%)</h4>

    Результат:

    hsl(0, 100%, 50%)

    hsl(240, 100%, 50%)

    hsl(147, 50%, 47%)

    hsl(300, 76%, 72%)

    hsl(39, 100%, 50%)

    hsl(248, 53%, 58%)

    В HTML вы можете указать цвета, используя значения HSL

    Saturation - Насыщенность

    Насыщенность можно описать как интенсивность цвета.

    100% - чистый цвет, без оттенков серого.

    50% - 50% серые, но вы все равно можете видеть цвет.

    0% полностью серого цвета, вы больше не можете видеть цвет.

    <h4 style="background-color:hsl(0, 100%, 50%);">hsl(0, 100%, 50%)</h4>
    <h4 style="background-color:hsl(0, 80%, 50%);">hsl(0, 80%, 50%)</h4>
    <h4 style="background-color:hsl(0, 60%, 50%);">hsl(0, 60%, 50%)</h4>
    <h4 style="background-color:hsl(0, 40%, 50%);">hsl(0, 40%, 50%)</h4>
    <h4 style="background-color:hsl(0, 20%, 50%);">hsl(0, 20%, 50%)</h4>
    <h4 style="background-color:hsl(0, 0%, 50%);">hsl(0, 0%, 50%)</h4>

    Результат:

    hsl(0, 100%, 50%)

    hsl(0, 80%, 50%)

    hsl(0, 60%, 50%)

    hsl(0, 40%, 50%)

    hsl(0, 20%, 50%)

    hsl(0, 0%, 50%)

    С цветами HSL меньшая насыщенность означает меньший цвет. 0% полностью серый.

    Lightness - Лёгкость (освещенность)

    Лёгкость (освещенность) цвета может быть описана как количество света, которое вы хотите дать цвету, где 0% означает отсутствие света (черный), 50% означает 50% света (ни темное, ни светлое). 100% означает полную яркость (белый).

    <h4 style="background-color:hsl(0, 100%, 0%);">hsl(0, 100%, 0%)</h4>
    <h4 style="background-color:hsl(0, 100%, 25%);">hsl(0, 100%, 25%)</h4>
    <h4 style="background-color:hsl(0, 100%, 50%);">hsl(0, 100%, 50%)</h4>
    <h4 style="background-color:hsl(0, 100%, 75%);">hsl(0, 100%, 75%)</h4>
    <h4 style="background-color:hsl(0, 100%, 90%);">hsl(0, 100%, 90%)</h4>
    <h4 style="background-color:hsl(0, 100%, 100%);">hsl(0, 100%, 100%)</h4>

    Результат:

    hsl(0, 100%, 0%)

    hsl(0, 100%, 25%)

    hsl(0, 100%, 50%)

    hsl(0, 100%, 75%)

    hsl(0, 100%, 90%)

    hsl(0, 100%, 100%)

    При использовании цветов HSL 0% легкость означает черный, а 100 - белый.

    Оттенки серого часто определяются путем установки оттенка и насыщенности на 0 и регулировки яркости от 0% до 100% для получения более темных / более светлых оттенков:

    <h4 style="background-color:hsl(0, 0%, 0%);">hsl(0, 0%, 0%)</h4>
    <h4 style="background-color:hsl(0, 0%, 24%);">hsl(0, 0%, 24%)</h4>
    <h4 style="background-color:hsl(0, 0%, 47%);">hsl(0, 0%, 47%)</h4>
    <h4 style="background-color:hsl(0, 0%, 71%);">hsl(0, 0%, 71%)</h4>
    <h4 style="background-color:hsl(0, 0%, 94%);">hsl(0, 0%, 94%)</h4>
    <h4 style="background-color:hsl(0, 0%, 100%);">hsl(0, 0%, 100%)</h4>

    Результат:

    hsl(0, 0%, 0%)

    hsl(0, 0%, 24%)

    hsl(0, 0%, 47%)

    hsl(0, 0%, 71%)

    hsl(0, 0%, 94%)

    hsl(0, 0%, 100%)

    Значение RGBA

    Значения цветов RGBA являются расширением значений цвета ,RGB с альфа-каналом, который определяет непрозрачность для цвета. Значение цвета RGBA определяется с помощью:

    rgba (red - красный, green - зеленый, blue - синий, alpha - альфа)

    Параметр alpha представляет собой число между 0.0 (полностью прозрачным) и 1.0 (вообще не прозрачным):

    <h4 style="background-color:rgba(255, 99, 71, 0);">rgba(255, 99, 71, 0)</h4>
    <h4 style="background-color:rgba(255, 99, 71, 0.2);">rgba(255, 99, 71, 0.2)</h4>
    <h4 style="background-color:rgba(255, 99, 71, 0.4);">rgba(255, 99, 71, 0.4)</h4>
    <h4 style="background-color:rgba(255, 99, 71, 0.6);">rgba(255, 99, 71, 0.6)</h4>
    <h4 style="background-color:rgba(255, 99, 71, 0.8);">rgba(255, 99, 71, 0.8)</h4>
    <h4 style="background-color:rgba(255, 99, 71, 1);">rgba(255, 99, 71, 1)</h4>

    Результат:

    rgba(255, 99, 71, 0)

    rgba(255, 99, 71, 0.2)

    rgba(255, 99, 71, 0.4)

    rgba(255, 99, 71, 0.6)

    rgba(255, 99, 71, 0.8)

    rgba(255, 99, 71, 1)

    Вы можете сделать прозрачные цвета, используя значение цвета RGBA

    Значение HSLA

    Значения цветов HSLA являются расширением значений цвета HSL с альфа-каналом, который определяет непрозрачность для цвета. Значение цвета HSLA указано с помощью:

    hsla (hue - оттенок, saturation - насыщенность, lightness - легкость, alpha - альфа)

    Параметр alpha представляет собой число между 0.0 (полностью прозрачным) и 1.0 (вообще не прозрачным):

    <h4 style="background-color:hsla(9, 100%, 64%, 0);">hsla(9, 100%, 64%, 0)</h4>
    <h4 style="background-color:hsla(9, 100%, 64%, 0.2);">hsla(9, 100%, 64%, 0.4)</h4>
    <h4 style="background-color:hsla(9, 100%, 64%, 0.4);">hsla(9, 100%, 64%, 0.4)</h4>
    <h4 style="background-color:hsla(9, 100%, 64%, 0.6);">hsla(9, 100%, 64%, 0.6)</h4>
    <h4 style="background-color:hsla(9, 100%, 64%, 0.8);">hsla(9, 100%, 64%, 0.8)</h4>
    <h4 style="background-color:hsla(9, 100%, 64%, 1);">hsla(9, 100%, 64%, 1)</h4>

    hsla(9, 100%, 64%, 0)

    hsla(9, 100%, 64%, 0.4)

    hsla(9, 100%, 64%, 0.4)

    hsla(9, 100%, 64%, 0.6)

    hsla(9, 100%, 64%, 0.8)

    hsla(9, 100%, 64%, 1)

    Вы можете делать прозрачные цвета, используя значение цвета HSLA.
