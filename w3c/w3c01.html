    1. HTML. Введение

    Что такое HTML?

    • HTML - это стандартный язык разметки для создания веб-страниц.
    • HTML обозначает язык разметки гипертекста;
    • HTML описывает структуру веб-страниц с использованием разметки;
    • HTML-элементы являются строительными блоками HTML-страниц;
    • Элементы HTML представлены тегами;
    • HTML-теги обозначают фрагменты контента, такие как «заголовок», «абзац», «таблица» и т.д.;
    • Браузеры не отображают теги HTML, но используют их для отображения содержимого страницы.
    Простой HTML-документ

    Пример:

    <!DOCTYPE html>
    <html>
    <head>
    <title>Название страницы</title>
    </head>
    <body>
    <h1>Первый заголовок</h1>
    <p>Первый параграф.</p>
    </body>
    </html>

    Разъяснения

    • Объявление <!DOCTYPE html> определяет документ как HTML5;
    • Элемент <html> является корневым элементом HTML-страницы;
    • Элемент <head> содержит метаинформацию о документе;
    • Элемент <title> указывает заголовок документа;
    • Элемент <body> содержит видимое содержимое страницы;
    • Элемент <h1> определяет большой заголовок;
    • Элемент <p> определяет абзац;

    HTML-теги

    HTML-теги - это имена элементов, окруженные угловыми скобками:

    <имя тега>Содержание здесь...</имя тега>

    HTML-теги обычно парные, как например <p> и </p>.

    Первый тег в паре - это начальный тег, второй тег - конечный тег. Конечный тег написан как начальный тег, но с косой чертой (слешем), вставленной перед именем тега.

    Начальный тег также называется открывающий тег, а конечный тег - закрывающий тег.

    Веб-браузеры

    Задача веб-браузера (Chrome, Firefox, Opera, Safari) - считывать HTML-документы и отображать их. Браузер не отображает теги HTML, но использует их для определения способа отображения документа:


    Отображение HTML-страницы в браузере

    Структура страницы HTML

    Ниже представлена визуализация структуры HTML-страницы:

    Примечание. В браузере отображается только контент внутри раздела <body>.


    <html>
    <head>
    <title>Название страницы</title>
    </head>
    <body>
    <h1>Это заголовок</h1>
    <p>Это параграф.</p>
    <p>Это другой параграф.</p>
    </body>
    </html>

    Тип документа <! DOCTYPE>

    Объявление <!DOCTYPE> представляет тип документа и позволяет браузерам правильно отображать веб-страницы.

    Он должен появляться только один раз, в верхней части страницы (перед любыми тегами HTML).

    Объявление <!DOCTYPE> не чувствительно к регистру.

    Объявление <!DOCTYPE> для HTML5:

    <!DOCTYPE html>
