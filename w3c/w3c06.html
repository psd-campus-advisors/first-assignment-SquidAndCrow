    6. HTML. Заголовки

    Заголовки определяются с тегами <h1> до <h6>. <h1> определяет наиболее важный заголовок (самого большого размера). <h6> определяет наименее важный заголовок (самого маленького размера).

    <h1>Heading 1</h1>

    <h2>Heading 2</h2>

    <h3>Heading 3</h3>

    <h4>Heading 4</h4>

    <h5>Heading 5</h5>

    <h6>Heading 6</h6>

    Примечание. Браузеры автоматически добавляют некоторое свободное пространство (отступ) до и после заголовка.

    Заголовки важны

    Поисковые системы используют заголовки для индексации структуры и содержимого веб-страниц. Пользователи просматривают страницы по заголовкам. Для отображения структуры документа важно использовать заголовки. Заголовки <h1> должны использоваться для основных заголовков, затем <h2> заголовки, затем менее важные <h3> и т.д.

    Примечание. Используйте заголовки HTML только для заголовков. Не используйте заголовки для создания БОЛЬШОГО текста или просто текста жирным шрифтом.

    Большие заголовки

    Каждый заголовок HTML имеет размер по умолчанию. Однако вы можете указать размер для любого заголовка с атрибутом style:

    <h1 style="font-size:60px;">Heading 1</h1>

    Горизонтальные разделители

    Тег <hr> определяет тематический разрыв на странице HTML и чаще всего отображается как горизонтальная линия. Элемент <hr> используется для разделения содержимого (или определения изменения) на HTML странице:

    <h1>Это заголовок 1</h1>
    <p>Это какой-то текст.</p>
    <hr>
    <h2>Это заголовок 2</h2>
    <p>Это какой-то другой текст.</p>
    <hr>

    Элемент HTML <head>

    Элемент HTML <head> (голова) не имеет ничего общего с заголовками HTML. Элемент <head> - это контейнер для метаданных. Метаданные HTML - это данные о документе HTML. Метаданные не отображаются на странице. Элемент <head> помещается между тегом <html> и тегом <body>:

    <DOCTYPE html>
    <html>
    <head>
    <title>Моя первая HTML-страница</title>
    <meta charset="UTF-8">
    </head>
    <body>
    <p>Элемент head содержит метаданные</p>
    <p>Метаданные - это данные о документе HTML</p>
    </body>
    </html>

    Примечание. Метаданные обычно определяют заголовок документа, набор символов, стили, ссылки, скрипты и другую метаинформацию.

    Как просмотреть исходный HTML-код страницы?

    Вы когда-нибудь видели веб-страницу и задавались вопросом: «Интересно, а как они это сделали?»

    Просмотреть исходный код HTML:

    Чтобы просмотреть исходный код HTML-страницы, щелкните правой кнопкой мыши на странице и выберите «View Page Source» / «Программный код страницы» (в Firefox) или «Просмотреть источник страницы» (в Chrome) или аналогичный в других браузерах. Откроется окно, содержащее исходный код HTML страницы.

    Просмотреть элемент HTML:

    Щелкните правой кнопкой мыши элемент (или пустую область) и выберите «Inspect» / «Исследовать» или «Inspect Element» / «Проверить элемент», чтобы увидеть, из каких элементов состоит страница (вы увидите как HTML, так и CSS). Вы также можете редактировать HTML или CSS на лету в раскрывающемся меню Элементы или стили.

    Проверьте себя с помощью упражнений!

    Ссылка на HTML-теги

    Ссылка на теги W3Schools содержит дополнительную информацию об этих тегах и их атрибутах. Вы узнаете больше о тегах и атрибутах HTML в следующих главах этого руководства.

    Тег Описание
    <html> Определяет корень HTML-документа
    <body> Определяет тело документа
    <head> Контейнер для всех элементов заголовка (название, скрипты, стили, метаинформация и т.д.)
    от <h1> до <h6> Определяет заголовки HTML
    <hr> Определяет тематическое изменение содержимого
