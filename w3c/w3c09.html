    9. HTML. Форматирование

    Пример:

    <p><b>Этот текст полужирный</b></p>
    <p><i>Этот текст написан курсивом</i></p>
    <p>Это <sub>подстрочный индекс</sub> и <sup>надстрочный индекс</sup></p>

    Результат:

    Этот текст полужирный

    Этот текст написан курсивом

    Это подстрочный индекс и надстрочный индекс

    Элементы форматирования HTML

    HTML также определяет специальные элементы для определения текста со специальным значением. HTML использует такие элементы, как <b> и <i> для форматирования вывода, например полужирный или курсивный текст.

    Элементы форматирования были разработаны для отображения специальных типов текста:

    • <b> - полужирный текст
    • <strong> - важный текст
    • <i> - курсивный текст
    • <em> - подчеркнутый текст
    • <mark> - маркированный текст
    • <smoll> - маленький текст
    • <del> - удаленный текст
    • <ins> - введенный текст
    • <sub> - текст подстрочный
    • <sup> - текст надстрочный

    Элементы <b> и <strong>

    Элемент HTML <b> определяет полужирный текст без лишней важности.

    <b>Этот текст полужирный</b>

    Результат:

    Этот текст полужирный


    Элемент HTML <strong> также делает текст полужирным, но при этом добавляет тексту «сильной» смысловой значимости.

    <strong>Этот текст выделен strong</strong>

    Результат:

    Этот текст выделен strong


    Элементы HTML <i> и <em>

    Элемент HTML <i> определяет курсивный текст без особого значения.

    <i>Этот текст написан курсивом</i>

    Результат:

    Этот текст написан курсивом


    Элемент HTML <em> определяет выделенный текст с добавленной смысловой значимостью.

    <em>Этот текст подчёркивается (имеет бОльшую значимость)</em>

    Результат:

    Этот текст подчёркивается (имеет бОльшую значимость)


    Примечание. Браузеры отображают <strong> как <b> (полужирным) и <em> как <i> (курсивом). Тем не менее, существует разница в значении этих тегов: <b> и <i> определяет жирный и курсивный текст, но теги <strong> и <em> означают, что текст является «важным».

    Элемент HTML <small>

    Элемент HTML <small> определяет уменьшенный текст:

    <h3>HTML <small>уменьшенный текст</small> форматирование</h3>

    Результат:

    HTML уменьшенный текст форматирование

    Элемент HTML <mark>

    Элемент HTML <mark> определяет особо отмеченный или выделенный текст (по умолчанию – желтым цветом):

    <h3>HTML <mark>Маркированное</mark> форматирование</h3>

    Результат:

    HTML Маркированное форматирование

    Элемент HTML <del>

    Элемент HTML <del> определяет удаленный (извлеченный) текст.

    <p>Мой любимый цвет <del>фиолетовый</del> красный.</p>

    Результат:

    Мой любимый цвет фиолетовый красный.

    Элемент HTML <ins>

    Элемент HTML <ins> определяет вставленный (подчёркнутый) текст.

    <p>Мой любимый <ins>цвет</ins> красный.</p>

    Результат:

    Мой любимый цвет красный.

    Элемент HTML <sub>

    Элемент HTML <sub> определяет подстрочный текст.

    <p>Это <sub>подстрочный</sub> текст.</p>

    Результат:

    Это подстрочный текст.

    Элемент HTML <sup>

    Элемент HTML <sup> определяет надстрочный текст.

    <p>Это <sup>надстрочный</sup> текст.</p>

    Результат:

    Это надстрочный текст.

    Элементы форматирования текста HTML

    Тег Описание
    <b> Определяет полужирный текст
    <em> Определяет выделенный текст
    <i> Определяет курсивный текст
    <small> Определяет уменьшенный текст
    <strong> Определяет важный текст
    <sub> Определяет подстрочный текст
    <sup> Определяет надстрочный текст
    <ins> Определяет подчёркнутый текст
    <del> Определяет удалённый текст
    <mark> Определяет маркированный/выделенный текст (по умолчанию желтым цветом)

    Проверьте себя с помощью упражнений!
