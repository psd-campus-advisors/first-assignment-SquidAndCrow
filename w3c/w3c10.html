    10. HTML. Цитаты и элементы цитирования

    Элемент Quotation (Цитата)

    <p>Это цитата из сайта WWF:</p>
    <blockquote cite="http://www.worldwildlife.org/who/index.html">
    В течение 50 лет WWF защищает будущее природы. Ведущая в мире природоохранная организация, WWF работает в 100 странах и поддерживается 1,2 миллиона членов в Соединенных Штатах и около 5 миллионов человек в мире.</blockquote>

    Элемент <q> для коротких цитат

    Элемент HTML <q> определяет короткую цитату. Браузеры обычно вставляют кавычки вокруг элемента <q>

    <p> Цель WWF: <q> построить будущее, в котором люди будут жить в гармонии с природой.</q></p>

    Элемент <blockquote> для цитат

    Элемент HTML <blockquote> определяет раздел, который цитируется из другого источника. Обычно для браузеров используются элементы отступов <blockquote>.

    <p>Это цитата из сайта WWF:</p>
    <blockquote cite="http://www.worldwildlife.org/who/index.html">
    В течение 50 лет WWF защищает будущее природы.
    Ведущая в мире природоохранная организация, WWF
    работает в 100 странах и поддерживается 1,2 миллиона членов
    в Соединенных Штатах и около 5 миллионов человек в мире.
    </blockquote>

    Элемент <abbr> для аббревиатуры

    Элемент HTML <abbr> определяет аббревиатуру или акроним. Маркировка аббревиатур может предоставить полезную информацию для браузеров, систем перевода и поисковых систем.

    <p><abbr title="Всемирная организация здравоохранения">ВОЗ</abbr> была образована в 1948 году.</p>

    Элемент <address> для контактной информации

    Элемент HTML <address> определяет контактную информацию (автора / владельца) документа или статьи. Элемент <address> обычно отображается курсивом. Большинство браузеров добавят разрыв строки до и после элемента.

    <address>
    Автор Вебмастер.<br>
    Посетите нас на:<br>
    Webmaster.com<br>
    а/с №123, Киев<br>
    Украина<br>
    </address>

    Элемент <cite> для названия работы

    Элемент HTML <cite> определяет название работы. Браузеры обычно отображают элементы <cite> курсивом.

    <p><cite>Умираючий гладіатор</cite> Тарас Шевченко. Написана в 1856 році.</p>

    Элемент <bdo> (Bi-Directional Override) для двунаправленного переопределения

    Элемент HTML <bdo> определяет двунаправленное переопределение. Элемент <bdo> используется для переопределения текущего направления текста:

    <bdo dir="rtl">Этот текст будет написан справа налево </bdo>

    Цитаты и элементы цитирования HTML

    Тег Описание
    <abbr> Определяет аббревиатуру или акроним
    <address> Определяет контактную информацию для автора/владельца документа
    <bdo> Определяет направление текста
    <blockquote> Определяет раздел, который цитируется из другого источника
    <cite> Определяет название работы
    <q> Определяет короткую встроенную цитату

    Проверьте себя с помощью упражнений!
