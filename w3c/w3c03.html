    3. HTML. Основные примеры

    Не беспокойтесь, если в этих примерах используются теги, которые вы не изучили. Вы узнаете о них в следующих главах.

    Документы HTML

    Все HTML-документы должны начинаться с объявления типа документа: <!DOCTYPE html>.

    Сам документ HTML начинается с <html> и заканчивается </html>.

    Видимая часть документа HTML находится между <body> и </body>.

    Пример:

    <!DOCTYPE html>
    <html>
    <body>
    <h1>Мой первый заголовок</h1>
    <p>Мой первый параграф.</p>
    </body>
    </html>

    Заголовки HTML

    Заголовки HTML определяются с тегами <h1> до <h6>.

    Тег <h1> определяет наиболее важный заголовок. Тег <h6> определяет наименее важный заголовок:

    Пример:

    <!DOCTYPE html>
    <html>
    <body>
    <h1>Это заголовок h1</h1>
    <h2>Это заголовок h2</h2>
    <h3>Это заголовок h3</h3>
    <h4>Это заголовок h4</h4>
    <h5>Это заголовок h5</h5>
    <h6>Это заголовок h6</h6>
    </body>
    </html>

    Результат:

    Это заголовок h1

    Это заголовок h2

    Это заголовок h3

    Это заголовок h4

    Это заголовок h5
    Это заголовок h6

    HTML-параграфы (абзацы)

    HTML-параграфы (абзацы) определяются тегом <p>:

    Пример:

    <!DOCTYPE html>
    <html>
    <body>
    <p>Это параграф.</p>
    <p>Это другой параграф.</p>
    </body>
    </html>

    Ссылки на HTML

    Ссылки HTML определяются с помощью тега <a>:

    Пример:

    <!DOCTYPE html>
    <html>
    <body>
    <a href="https://www.w3schools.com">Это ссылка</a>
    </body>
    </html>

    Пункт назначения ссылки указан в атрибуте href. Атрибуты используются для предоставления дополнительной информации об элементах HTML.

    Изображения HTML

    Изображения HTML определяются тегом <img>.

    Исходный файл (src), альтернативный текст (alt), ширина и высота предоставляются в качестве атрибутов:

    <!DOCTYPE html>
    <html>
    <body>
    <img src="https://www.w3schools.com/html/w3schools.jpg" alt="W3Schools.com" width="104" height="142">
    </body>
    </html>
