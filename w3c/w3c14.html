14. HTML ссылки

Ссылки присутствуют на большинстве веб-страниц. Ссылки позволяют пользователям кликать по ним и переходить со страницы на страницу, с одного сайта на другой.

HTML ссылки – это гиперссылки. Вы можете нажать на ссылку и перейти на другой документ (другую веб-страницу). Когда вы наводите мышь на ссылку, стрелка мыши обычно превращается в маленькую ладонь.

Примечание. Ссылка не обязательно должна быть текстовой. Это также может быть изображение или любой другой элемент HTML.

HTML ссылки. Синтаксис

В HTML ссылки определяются тегом <a> (сокращенно от слова anchor – якорь):

Пример:

<a href="url">Текст ссылки</a>
<a href="https://www.w3schools.com/html/">Учебник HTML</a>

Атрибут href указывает адрес получателя ссылки (https://www.w3schools.com/html/).

Текст ссылки - видимая часть. Нажав на текст ссылки, вы перейдёте по указанному адресу.

Примечание. Без косой черты (слеша) на адресах подпапки вы можете создать на сервере два запроса. Многие серверы автоматически добавят косую черту в адрес и затем создадут новый запрос.

Локальные ссылки

В приведенном выше примере используется абсолютный URL (полный веб-адрес).

Локальная ссылка (ссылка на тот же веб-сайт) указана с относительным URL (без http: // www...).

Пример:

<a href="html_images.asp">HTML Images</a>

Цвета HTML ссылок

По умолчанию ссылка будет отображаться (во всех браузерах) так:

  • Не посещенная ссылка подчеркнута и синего цвета;
  • Посещенная ссылка подчеркнута и фиолетового цвета;
  • Активная ссылка подчеркнута и красного цвета.

Вы можете изменить цвета по умолчанию, используя CSS:

Пример:

<style>
a:link {
  color: green;
  background-color: transparent;
  text-decoration: none;
}
a:visited {
  color: pink;
  background-color: transparent;
  text-decoration: none;
}
a:hover {
  color: red;
  background-color: transparent;
  text-decoration: underline;
}
a:active {
  color: yellow;
  background-color: transparent;
  text-decoration: underline;
}
</style>

где:
a:link - обычная ссылка;
a:visited - посещенная ссылка;
a:hover - ссылка при наведении на неё мышки;
a:active - активная (открытая) ссылка.

Целевой атрибут (attribute target) в HTML ссылках

Атрибут target указывает, где можно открыть связанный документ.

Атрибут target может иметь одно из следующих значений:

  • _blank - открывает связанный документ в новом окне или вкладке;
  • _self - открывает связанный документ в том же окне/вкладке, когда он был нажат (работает в браузерах по умолчанию);
  • _parent - открывает связанный документ в родительском фрейме;
  • _top - открывает связанный документ во всё открытое окно;
  • framename - открывает связанный документ в именованном фрейме.

В этом примере откроется связанный документ в новом окне / вкладке браузера:

Пример:

<a href="https://www.w3schools.com/" target="_blank">Посетите W3Schools!</a>

Совет. Если ваша веб-страница заблокирована во фрейме, вы можете использовать target="_top" для выхода из фрейма:

Пример:

<a href="https://www.w3schools.com/html/" target="_top">HTML5 инструкцию!</a>

Изображение как ссылка

Обычно изображения используются как ссылки:

Пример:

<a href="default.asp">
<img src="smiley.gif" alt="HTML tutorial" style="width:42px;height:42px;border:0;">
</a>

Примечание: граница указывается ноль (border:0;), чтобы IE9 (и ранее) не отображал границу вокруг изображения (когда изображение является ссылкой).

Атрибут title в ссылках

Атрибут title указывает дополнительную информацию об элементе. Информация чаще всего отображается в виде текста всплывающей подсказки, когда курсор находится над элементом.

Пример:

<a href="https://www.w3schools.com/html/" title="Перейти в раздел HTML на W3Schools">Посетите HTML руководство.</a>

HTML ссылки, создающие закладку

HTML-закладки используются, чтобы позволить читателям перейти к определенным частям веб-страницы. Закладки могут быть полезны, если ваша веб-страница очень длинная. Чтобы сделать закладку, вы должны сначала создать закладку, а затем добавить к ней ссылку. При щелчке по ссылке страница будет прокручиваться до местоположения с закладкой.

Пример:

Сначала создайте закладку с атрибутом id:

<h2 id="C4">глава 4</h2>

Затем добавьте ссылку на закладку («Перейти к главе 4»), с той же страницы:

<a href="#C4">Перейти к главе 4</a>

Или добавьте ссылку на закладку («Перейти к главе 4»), с другой страницы:

<a href="html_demo.html#C4">Перейти к главе 4</a>

Внешние пути

На внешние страницы можно ссылаться с полным URL-адресом или с указанием пути к текущей веб-странице. В этом примере для ссылки на веб-страницу используется полный URL-адрес:

Пример:

<a href="https://www.w3schools.com/html/default.asp">HTML инструкция</a>

В этом примере ссылка на страницу, расположенную в папке html на текущем веб-сайте:

Пример:

<a href="/html/default.asp">HTML инструкция</a>

Этот пример ссылается на страницу, расположенную в той же папке, что и на текущей странице:

<a href="default.asp">HTML tutorial</a>

Также ссылку можно связать с почтовым сообщением mailto (будет работать, только если у вас установлена электронная почта - email).

Это ссылка на email:

<a href="mailto:someone@example.com?Subject=Hello%20again" target="_top">Отправить почту</a>

Примечание. Пространства между словами следует заменить на %20, чтобы убедиться, что браузер правильно отобразит текст.

Или можно просто сделать ссылку с указанием email:

<a href="mailto:user@example.com target="_top">Написать сообщение</a>

И откроется почтовый клиент, который установлен по умолчанию. (На Windows, где установлен пакет Microsoft Office, обычно запускается Outlook).

Ссылка на элемент с указанным идентификатором id

Как связать элемент с указанным идентификатором внутри страницы (атрибут name не поддерживается в HTML5).

<h2 id="top">Начало страницы</h2>
<p>много текста</p>
<p>…много текста</p>
<p>…много текста</p>
<p>…много текста</p>
<p>…много текста</p>
<a href="#top">Перейти на начало!</a>

Нажав на ссылку "Перейти на начало!" вас автоматически вернёт на начало страницы или к тому месту, где прикреплён id.

Ссылка на JavaScript

Можно также добавить JavaScript-код внутри атрибута href:

<a href="javascript:alert('Hello World!');">Выполнить JavaScript</a>

Подробнее о путях файлов читайте в разделе «Пути файлов HTML».

Различия между HTML 4.01 и HTML5

В HTML 4.01 тег <a> может быть либо гиперссылкой, либо якорем. В HTML5 тег <a> всегда является гиперссылкой, но если он не имеет атрибута href, он является только заполнителем для гиперссылки.

HTML5 имеет несколько новых атрибутов, а некоторые атрибуты HTML4.01 больше не поддерживаются.

Настройки CSS по умолчанию для ссылки

В большинстве браузеров будет отображаться элемент <a> со следующими значениями по умолчанию:

a:link, a:visited {
  color: (internal value);
  text-decoration: underline;
  cursor: auto;
}
a:link:active, a:visited:active {
  color: (internal value);
}

При необходимости данные настройки можно изменить!

Краткое содержание главы

  • Используйте элемент <a> для определения ссылки;
  • Используйте атрибут href для определения адреса ссылки;
  • Используйте целевой атрибут (target), чтобы определить, где открыть связанный документ;
  • Используйте элемент <img> (внутри <a>), чтобы использовать изображение в качестве ссылки;
  • Используйте атрибут id (id="значение") для определения закладок на странице;
  • Используйте ссылку href (href="#значение") для ссылки на закладку.

Атрибуты

= Новый в HTML5.

Атрибут Значение Описание
charset char_encoding Не поддерживается в HTML5.
Задает набор символов связанного документа
coords coordinates Не поддерживается в HTML5.
Указывает координаты ссылки
download filename Указывает, что цель будет загружаться, когда пользователь нажимает на гиперссылку
href URL Указывает URL-адрес страницы, на которую ссылается ссылка
hreflang language_code Задает язык связанного документа
media media_query Определяет, какой носитель/устройство привязан к документу для оптимизации
name section_name Не поддерживается в HTML5. Вместо этого используйте глобальный атрибут id.
Задает имя привязки (якоря)
rel alternate, author, bookmark, external, help, license, next, nofollow, noreferrer, noopener, prev, search, tag Задает связь между текущим документом и связанным документом
rev text Не поддерживается в HTML5.
Задает взаимосвязь между связанным документом и текущим документом
shape default, rect, circle, poly Не поддерживается в HTML5.
Определяет форму ссылки
target _blank, _parent, _self, _top, framename Определяет, где можно открыть связанный документ
type media_type Задает тип носителя связанного документа
