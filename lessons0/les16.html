Блочные и встроенные (строчные) элементы в HTML

Каждый элемент HTML имеет отображаемое значение по умолчанию в зависимости от типа элемента. Значение по умолчанию для большинства элементов является блочным или встроенным.

Блочные элементы

Блочный элемент всегда начинается с новой строки и занимает всю доступную ширину на странице (растягивается влево и вправо, насколько это возможно).

Элемент <div>

Элемент <div> (от англ. division — раздел, подразделение) является универсальным блочным элементом и предназначен для группирования элементов документа с целью изменения вида содержимого через стили. Для этого добавляется атрибут class или id с именем класса или идентификатора.

Синтаксис:

<div> … </div>

Закрывающий тег обязателен.

Пример:

<div>Hello</div>
<div>World</div>

Элемент DIV является блочным элементом и будет начинаться с новой строки.

Элемент <div> часто используется в качестве контейнера для других элементов HTML. Элемент <div> не имеет обязательных атрибутов, но стиль, класс и идентификатор являются общими. При использовании вместе с CSS элемент <div> можно использовать для стилей блоков содержимого:

Пример:

<div style="background-color:black;color:white;padding:20px;">
<h2>Лондон</h2>
<p>Лондон является столицей Англии. Это самый густонаселенный город в Соединенном Королевстве, с населением более 13 миллионов человек.</p>
</div>

Блочные элементы в HTML:

<address> <article> <aside> <blockquote> <canvas> <dd> <div> <dl> <dt> <fieldset> <figcaption> <figure> <footer> <form> <h1>-<h6> <header> <hr> <li> <main> <nav> <noscript> <ol> <output> <p> <pre> <section> <table> <tfoot> <ul> <video>

Встроенные (строчные) элементы

Встроенный (строчный) элемент не начинается с новой строки и занимает столько ширины, сколько необходимо. Например, это встроенный элемент <span> внутри параграфа (абзаца).

Элемент <span>

Универсальный строчный элемент <span> (от англ. span — охватывать) предназначен для выделения отдельных строк, символов или других строчных элементов для дальнейшего изменения их оформления с помощью стилей.

Синтаксис:

<span> ... </span>

Закрывающий тег обязателен.

Пример:

<span>Hello<span> <span>World</span>

Элемент SPAN является встроенным (строчным) элементом и не будет начинаться с новой строки. Элемент <span> часто используется как контейнер для некоторого текста. Элемент <span> не имеет обязательных атрибутов, но стиль, класс и идентификатор являются общими. При использовании вместе с CSS элемент <span> можно использовать для стилизации частей текста:

Пример:

<h1>Мой <span style="color:red">важный</span> Заголовок</h1>

Встроенные (строчные) элементы в HTML:

<a> <abbr> <acronym> <b> <bdo> <big> <br> <button> <cite> <code> <dfn> <em> <i> <img> <input> <kbd> <label> <map> <object> <q> <samp> <script> <select> <small> <span> <strong> <sub> <sup> <textarea> <time> <tt> <var>

<span> Определяет строчный раздел в документе
