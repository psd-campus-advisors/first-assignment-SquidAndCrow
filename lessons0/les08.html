Стили в HTML

HTML-теги - это контейнеры для текста или других элементов на веб-странице. Но сами теги служат для выводы информации и, как правило, не имеют визуального оформления. Если бы все веб-страницы были написаны только лишь с помощью одних HTML-тегов, то они были бы слишком однообразны и выглядели бы практически одинаково - с одним и тем же шрифтом, отступами, линиями и т.д. И, вообщем-то так раньше и было, когда веб-разметка только зарождалась. Но с развитием веб-технологий всё изменялось и усовершенствовалось. Однообразные текстовые веб-страницы стали приобретать более красочный вид с различным визуальным оформлением. С каждой новой спецификацией (стандартом) HTML-разметки визуальных эффектов становилось всё больше. К HTML-тегам стали применять уже и стилевое оформление, которое постепенно сформировалось в каскадные таблицы стилей - CSS, со своим собственным синтаксисом.

Если в первых ранних спецификациях HTML стилевое оформление прописывалось непосредственно в самом HTML-коде, то уже позже описание стилей стали выносить в отдельный файл, который подключается к HTML-документу с помощью определённого тега.

Сейчас используется три разновидности использования стилей в HTML-документе:

  1. Встроенный - с помощью атрибута style в элементах HTML;
  2. Внутренний - с помощью элемента <style> в разделе головы страницы <head> (встроенная таблица стилей);
  3. Внешний - с помощью подключения внешнего файла каскадной таблицы стилей (CSS).

Подробнее о каскадной таблице стилей я расскажу в следующих уроках. А пока в качестве примеров будем использовать встроенные стили с помощью атрибута style.

HTML атрибут style

Настройка стиля элемента HTML может быть выполнена с помощью атрибута style. Атрибут style имеет следующий синтаксис:

<tagname style="property:value;">

где
tagname - название тега,
property - свойство CSS,
value - значение CSS.

Пример 8.1: Использование стилей для каждого параграфа

<p style="font-size:1.9em; color:red">Я красный большой текст</p>
<p style="font-size:1.5em; color:blue">Я синий средний текст</p>
<p style="font-size:.9em; color:green">Я зелёный маленький текст</p>
Посмотреть результат!

Цвет фона страницы

Свойство background-color определяет цвет фона для элемента HTML.

В этом примере задается цвет фона для страницы:

Пример 8.2:

<body style="background-color:powderblue;">
<h1>Это заголовок</h1>
<p>Это параграф.</p>
</body>
Посмотреть результат!

Цвет фона страницы в этом примере будет голубым.

Цвет текста

Свойство color определяет цвет текста для двух элементов HTML - заголовка и параграфа:

Пример 8.3:

<h1 style="color:blue;">Это заголовок синего цвета</h1>
<p style="color:red;">Это параграф красного цвета.</p>
Посмотреть результат!

HTML-шрифты

Свойство font-family определяет семейство шрифта, который будет использоваться для элементов HTML - заголовка и параграфа:

Пример 8.4:

<h1 style="font-family:verdana;">Это заголовок</h1>
<p style="font-family:courier;">Это параграф.</p>
<p style="font-family:comic sans ms;">Это параграф.</p>
Посмотреть результат!

Размер текста (шрифта) HTML

Свойство font-size определяет размер текста (шрифта) для элементов HTML - заголовка и параграфа:

Пример 8.5:

<h1 style="font-size:200%;">Это заголовок</h1>
<p style="font-size:160%;">Это параграф.</p>
<p style="font-size:18px;">Это параграф.</p>
<p style="font-size:2em;">Это параграф.</p>
Посмотреть результат!

При этом размер текста можно задавать различными способами: с помощью пикселей, процентов и масштабируемой единицы (em).

Выравнивание текста HTML

Свойство text-align определяет горизонтальное выравнивание текста для элемента HTML:

Пример 8.6:

<h1 style="text-align:center;">Центрированный заголовок</h1>
<p style="text-align:center;">Центрированный параграф.</p>
<p style="text-align:left;">Выровненный по левому краю (по умолчанию).</p>
<p style="text-align:right;">Выровненный по правому краю.</p>
<p style="text-align:justify;">Выровненный по ширине (нужно много текста).</p>
Посмотреть результат!

Задавать стиль элементу также можно, перечислив различные свойства через точку с запятой.

Пример 8.7:

<body style="background-color:yellow;">
<h1 style="text-align:center; color:blue;">Главный заголовок</h1>
<p style="text-align:right; color:green; font-family:gabriola; font-size:20px;">Работа - не волк, в лес не убежит!<br>(русская народная мудрость)</p>
<p style="text-align:justify; color:indigo; font-family:calibri; font-size:1.5em;">А здесь много-много всякого текста первого параграфа...</p>
</body>
Посмотреть результат!

Проверьте себя с помощью упражнений!

Форматирование текста в HTML

Форматирование текста - это средства его изменения. Например, изменение начертания шрифта (жирный, курсивом, подчеркнутый и т.д.) и использование различных эффектов, позволяющих менять вид текста.

Пример 8.8:

<p><b>Этот текст полужирный</b></p>
<p><i>Этот текст написан курсивом</i></p>
<p>Это <sub>подстрочный индекс</sub> и <sup>надстрочный индекс</sup></p>
Посмотреть результат!

Результат:

Этот текст полужирный

Этот текст написан курсивом

Это подстрочный индекс и надстрочный индекс

Элементы форматирования HTML

HTML также определяет специальные элементы для определения текста со специальным значением.

Элементы форматирования текста в HTML постоянно меняются. В каждой новой спецификации HTML некоторые теги для форматирования текста удаляются, а новые появляются. Хотя в самих браузерах старые (удалённые из спецификации) теги обычно продолжают работать. Но при этом не рекомендуются для использования (например, тег <u>). На данный момент актуальными являются следующие теги:

  • <b> - полужирный текст
  • <strong> - полужирный (логически выделенный) текст
  • <i> - курсивный текст
  • <em> - курсивный (логически выделенный) текст
  • <mark> - маркированный (выделенный цветом) текст
  • <small> - маленький текст (меньший на единицу от обычного текста)
  • <del> - удаленный текст (перечёркивается)
  • <ins> - вставленный текст (выделенный подчеркиванием)
  • <sub> - текст подстрочный
  • <sup> - текст надстрочный

Элементы <b> и <strong>

Элемент HTML <b> определяет полужирный текст без лишней важности.

<b>Этот текст полужирный</b>

Результат:

Этот текст полужирный


Элемент HTML <strong> также делает текст полужирным, но при этом добавляет тексту «сильной» смысловой значимости. Относится к тегам логической разметки.

<strong>Этот текст выделен strong</strong>

Результат:

Этот текст выделен strong


Элементы HTML <i> и <em>

Элемент HTML <i> определяет курсивный текст без особого значения.

<i>Этот текст написан курсивом</i>

Результат:

Этот текст написан курсивом


Элемент HTML <em> определяет выделенный курсивом текст с добавленной смысловой значимостью. Относится к тегам логической разметки.

<em>Этот текст выделяется курсивом (имеет бОльшую значимость)</em>

Результат:

Этот текст выделяется курсивом (имеет бОльшую значимость)


Примечание. Браузеры отображают <strong> как <b> (полужирным) и <em> как <i> (курсивом). Тем не менее, существует разница в значении этих тегов: <b> и <i> определяет жирный и курсивный текст, но теги <strong> и <em> означают, что текст является «важным» (более значимым).

Элемент HTML <small>

Элемент HTML <small> определяет уменьшенный текст (по сравнению из текстом по умолчанию):

<h3>HTML <small>уменьшенный текст</small> форматирование</h3>

Результат:

HTML уменьшенный текст форматирование

Элемент HTML <mark>

Элемент HTML <mark> определяет особо отмеченный или выделенный текст (по умолчанию – желтым цветом):

<h4>HTML <mark>Маркированное</mark> форматирование</h4>

Результат:

HTML Маркированное форматирование

Элемент HTML <del>

Элемент HTML <del> определяет удаленный (извлеченный) текст. Используется для выделения текста, удаленного из документа. Отображается перечеркнутым. Для тега доступны следующие атрибуты: cite, datetime.

<p>Мой любимый цвет <del>фиолетовый</del> красный.</p>

Результат:

Мой любимый цвет фиолетовый красный.

Элемент HTML <ins>

Элемент HTML <ins> определяет вставленный текст, выделяя подчёркиванием. Для тега доступны следующие атрибуты: cite, datetime.

<p>Мой любимый <ins>цвет</ins> красный.</p>

Результат:

Мой любимый цвет красный.

Элемент HTML <sub>

Элемент HTML <sub> определяет подстрочный текст. Используется для создания нижних индексов. Сдвигает текст ниже уровня строки, уменьшая его размер.

<p>Это <sub>подстрочный</sub> текст.</p>

Результат:

Это подстрочный текст.

Элемент HTML <sup>

Элемент HTML <sup> определяет надстрочный текст. Используется для создания степеней. Сдвигает текст выше уровня строки, уменьшая его размер.

<p>Это <sup>надстрочный</sup> текст.</p>

Результат:

Это надстрочный текст.

Элементы <sub> и <sup> обычно используются для написания различных формул.

Например:

<p>Формула воды: H<sub>2</sub>O;</p>
<p>Формула Эйнштейна: e = mc<sup>2</sup>.</p>

Результат:

Формула воды: H2O; формула Эйнштейна: e=mc2.
