Определение таблицы HTML

Таблица в HTML определяется тегом <table> (с англ. - таблица).

Каждая строка таблицы определяется тегом <tr> (table row). Заголовок таблицы определяется тегом <th> (table header). По умолчанию заголовки таблицы выделены жирным шрифтом и центрированы. Ячейка (в которой обычно содержатся какие-либо данные) определяется тегом <td> (table data).

Заголовок <th> Заголовок <th> Заголовок <th>
ячейка <td> ячейка <td> ячейка <td>
ячейка <td> ячейка <td> ячейка <td>
ячейка <td> ячейка <td> ячейка <td>

Строка <tr> (в данной таблице выделена цветом) содержит в себе ячейки (в т.ч. и ячейки из заголовками).

<TABLE>

<table> (от англ. table — таблица) служит контейнером для элементов, определяющих содержимое таблицы. Любая таблица состоит из строк и ячеек, которые задаются с помощью элементов <tr> (строка) и <td> (ячейка). Внутри <table> допустимо использовать следующие элементы: <caption>, <col>, <colgroup>, <thead>, <tbody>, <tfoot>, <tr>, <th>, <td>.

Синтаксис:

<table>
<tr>
<td>...</td>
</tr>
</table>

Закрывающий тег обязателен.

<TR>

Элемент <tr> (от англ. table row — строка таблицы) служит контейнером для создания строки таблицы. Каждая ячейка в пределах такой строки устанавливается с помощью элемента <th> или <td>.

Синтаксис:

<table>
<tr>
<td>...</td>
</tr>
</table>

Закрывающий тег не обязателен.

<TH>

Элемент <th> (от англ. table header cell — ячейка заголовка таблицы) предназначен для создания одной ячейки таблицы, которая обозначается как заголовочная. Текст в такой ячейке отображается браузером по умолчанию жирным шрифтом и выравнивается по центру. Элемент <th> должен размещаться внутри контейнера <tr>, который в свою очередь располагается внутри <table>.

Синтаксис:

<table>
<tr>
<th>...</th>
</tr>
</table>

Закрывающий тег не обязателен.

<TD>

Элемент <td> - предназначен для создания одной ячейки таблицы. Элемент <td> (от англ. table data cell — ячейка с данными таблицы) должен размещаться внутри контейнера <tr>, который в свою очередь располагается внутри <table>.

Синтаксис:

<table>
<tr>
<td>...</td>
</tr>
</table>

Закрывающий тег не обязателен.

Атрибуты для тегов <TH> и <TD>:

abbr - краткое описание содержимого ячейки.

Текст внутри abbr браузерами не отображается и предназначен в первую очередь для доступа людей к содержанию с помощью вспомогательных технологий вроде речевых браузеров. Значение представляет сокращённую версию текста внутри элемента <th>. Хотя abbr никак не выводится на странице, браузеры распознают этот атрибут и могут получить к нему доступ через скрипты.

Синтаксис:

<th abbr="текст">...</th>

colspan - устанавливает число ячеек, которые должны быть объединены по горизонтали.

Синтаксис:

<th colspan="число">...</th>

rowspan - объединяет вертикальные ячейки. Этот атрибут имеет смысл для таблиц, состоящих из нескольких строк.

Синтаксис:

<th rowspan="число">...</th>

headers - позволяет связать ячейки таблицы с заголовками в речевых браузерах. Этот атрибут предназначен для повышения доступности таблицы пользователям речевых браузеров, в обычных браузерах результат применения атрибута headers не заметен. Для связывания ячеек между собой, ячейке в <th> пишется атрибут id, а в другой ячейке — атрибут headers со значением, совпадающим со значением id.

Синтаксис:

<th id="идентификатор">...</th>
<th headers="идентификатор">...</th>

Атрибуты для тега <td> аналогичны атрибутам для тега <th>.

Пример 14.1:

<table style="width:100%">
<tr>
<th>Firstname</th>
<th>Lastname</th>
<th>Age</th>
</tr>
<tr>
<td>Jill</td>
<td>Smith</td>
<td>50</td>
</tr>
<tr>
<td>Eve</td>
<td>Jackson</td>
<td>94</td>
</tr>
<tr>
<td>John</td>
<td>Doe</td>
<td>80</td>
</tr>
</table>

Таблица HTML. Добавление границы

Если вы не укажете границу для таблицы, она будет отображаться без границ. Граница устанавливается с использованием свойства border:

Пример 14.2:

table, th, td {
   border: 1px solid black;
}
Посмотреть результат!

Не забудьте определить границы как для таблицы, так и для ячеек таблицы.

Таблица HTML. Свернутые границы

Если вы хотите, чтобы границы отображались одинарной линией, добавьте свойство CSS border-collapse.

Пример 14.3:

table, th, td {
   border: 1px solid black;
   border-collapse: collapse;
}
Посмотреть результат!

Таблица HTML. Добавление отступа (padding) в ячейках

Отступ (padding) внутри ячеек определяет пространство между содержимым ячейки и ее границами. Если вы не укажете padding ячейки таблицы, содержимое будет отображаться без отступа, т.е. прилипать к границам ячеек. Чтобы установить отступ, используйте свойство padding в CSS:

Пример 14.4:

th, td {
   padding: 15px;
}

Таблица HTML. Заголовки с левым выравниванием

По умолчанию заголовки таблицы выделены жирным шрифтом и центрированы. Чтобы выровнять по горизонтали заголовки таблицы, используйте CSS свойство text-align:

Пример 14.5:

th {
   text-align: left;
}
Посмотреть результат!

Таблица HTML. Добавление интервала между границами

Межграничный интервал определяет пространство между ячейками. Чтобы установить интервал между границами для таблицы, используйте CSS свойство border-spacing:

Пример 14.6:

table {
   border-spacing: 5px;
}

Примечание. Если таблица имеет границы collapsed, интервал между границами никак не влияет.

Таблица HTML. Ячейки, которые охватывают множество столбцов

Таблица HTML. Добавление заголовка таблицы (подписи к таблице)

Чтобы добавить подпись к таблице, используйте тег <caption>:

Пример 14.9:

<th>Экономия</th>

</tr>

Особый стиль для одной таблицы

Чтобы определить особый стиль для отдельной таблицы, добавьте в таблицу атрибут id:

Пример 14.10:

<table id="t01">
<tr>
<th>Firstname</th>
<th>Lastname</th>
<th>Age</th>
</tr>
<tr>
<td>Eve</td>
<td>Jackson</td>
<td>94</td>
</tr>
</table>

Теперь вы можете определить особый стиль для этой таблицы:

table#t01 {
width: 100%;
background-color: #f1f1c1;
}

table#t01 tr:nth-child(even) {

Как было раньше?

Раньше, до спецификации HTML5 (в версиях HTML3, HTML4.01), таблицы очень активно использовались для написания целых веб-страниц, даже был такой "табличный стиль" создания веб-страниц. Но сейчас из-за требований адаптивности веб-сайтов и отображения на различных устройствах, табличный стиль оформления веб-страницы стал неактуальным, т.к. при такой вёрстке не все элементы удаётся разместить правильно и само восприятие информации на таких страницах при использовании различных гаджетов стало не очень удобным. Поэтому сейчас, в спецификации HTML5 таблицы используются только в качестве самих таблиц с данными, и не используются для создания всей структуры страницы, как это было раньше. Хотя практически все браузеры по-прежнему считывают все теги и атрибуты (в т.ч. и устаревшие), которые использовались при вёрстке веб-страниц в более ранних версиях HTML.

Для более наглядного примера давайте создадим простую веб-страницу, свёрстаную с помощью таблицы. Пусть на нашей странице будет шапка (header), подвал (footer) и основная часть, состоящая из левого сайдбара (left-sidebar), центральной части из содержимым (main) и правого сайдбара (right-sidebar).

Открываем HTML-редактор и создаём стандартную структуру страницы. Внутри тела страницы (в body) будет, как я уже сказал, header, main и footer. Теперь необходимо каждую из этих составляющих поместить внутри таблицы, которая и будет каркасом веб-страницы. В первой строке (tr) таблицы размещаем одну ячейку (td) с блоком header (HEADER). Во сторой строке размещаем три ячейки: первая ячейка - левый сайдбар (LEFT-SIDEBAR), вторая ячейка - центральная часть из основным содержимым (MAIN), и третья ячейка - правый сайдбар (RIGHT-SIDEBAR). В третьей строке также будет одна ячейка, в которой разместим блок FOOTER.

Обновим страницу в браузере и увидим содержимое блоков, которые пока ещё никак не отображены в самой таблице. Для того, чтобы увидеть саму таблицу, необходимо в теге table прописать атрибут border, т.е. границу, с указанием толщины в пикселях. В данном случае толщина границы будет в один пиксель. Обновим страницу в браузере и теперь видим саму таблицу. При этом размеры таблицы по умолчанию устанавливаются такие, чтобы в ней помещалось содержимое. Поэтому давайте зададим ширину таблицы 100%. В результате она растянется на всю ширину экрана (независимо от разрешения). Но мы видим что две ячейки - header и footer - не растягиваются на всю ширину таблицы. Для этого необходимо задать для каждой из этих ячеек атрибут colspan с указанием числа ячеек, на ширину которых они должны быть растянуты. В данном случае на 3 ячейки. Сохраним и обновим страницу. Видим, что теперь все ячейки растянуты на всю ширину таблицы. Но при этом ширина их произвольная. Давайте установим определённый размер для каждой из ячеек. Это можно сделать как в пикселях - фиксированный размер без возможности изменения, так и в процентах - гибкий размер, который будет изменяться в зависимости от размера окна браузера или разрешения экрана. Для левого сайдбара - 20%, для основного содержимого MAIN - 60%, для правого сайдбара - 20%. Имейте в виду, что проценты для ячеек могут быть разными, но в сумме должно быть 100%! Сохраним и обновим страницу. Видим, что размеры ячеек поменялись. Теперь выровняем по центру содержимое ячеек HEADER и FOOTER с помощью атрибута align. Сохраним и обновим страницу. Обычно блок HEADER содержит название сайта, которое оборачивается в теги h1. Аналогично обернём в тег h2 заголовок основного содержимого MAIN, а также выравняем его по центру страницы с помощью атрибута align и значения center. Сохраним и обновим страницу. Мы видим, как изменился внешний вид таблицы. Как вы заметили, по умолчанию содержимое ячеек таблицы выравнивается по горизонтали по левому краю, а по вертикали - по центру ячейки. Поэтому давайте зададим ещё выравнивание содержимого ячейки по вертикали, прижав его к верху с помощью атрибута valign со значением top (вверх) для каждой из ячеек - левого сайдбара, основного содержимого и правого сайдбара. Затем для наглядности заполним основную часть (MAIN) содержимым - произвольным текстом. Обновим страницу.

Границы таблицы по умолчанию отображаются как двойная сплошная линия. Но их можно сделать одинарными с помощью атрибута cellspacing со значением ноль. Также мы можем установить отступ между содержимым и границами таблицы в пикселях с помощью атрибута cellpadding, например, в 20 пиксель. Обновим страницу.

Теперь для большей наглядности зададим цвета для каждой из ячеек таблицы.

<body>
<table border="1" cellspacing="0" cellpadding="20" width="100%">
<tr>
<header>
<td colspan="3" align="center" bgcolor="#ccc"><h1>HEADER</h1></td>
</header>
</tr>
<tr>
<td width="20%" valign="top" bgcolor="#eee">LEFT-SIDEBAR</td>
<td width="60%" valign="top"><h2 align="center">MAIN</h2>
<main>
<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.</p>
</main>
</td>
<td width="20%" valign="top" bgcolor="#eee">RIGHT-SIDEBAR</td>
</tr>
<tr>
<footer>
<td colspan="3" align="center" bgcolor="#ccc">FOOTER</td>
</footer>
<tr>
</table>
</body>

Как видно, код в общем-то небольшой. И при этом мы создали практически готовый шаблон страницы. Но при таком табличном стиле создания веб-страницы самым большим недостатком является отстутствие её адаптивности. Т.е. при разных разрешениях экрана такая страница будет не адаптивной и некоторые элементы на странице будут находится за пределами видимой части экрана и будет появляться горизонтальная полоса прокрутки (что не очень удобно). Поэтому сейчас не рекомендуется использовать такой табличный метод верстки HTML-страниц, а всё делать с помощью CSS - каскадных таблиц стилей.

Кроме того, данный код не пройдет проверку на валидность и соответствие спецификации HTML5. Валидатор сообщит, что практически все атрибуты, используемые при написании HTML-кода в данной таблице, являются устаревшими и рекомендуется использовать CSS.


В этом видеоуроке вы узнаете о том, как не надо верстать веб-страницы с помощью таблиц.

Краткое содержание главы

  • Используйте элемент HTML <table> для определения таблицы;
  • Используйте элемент HTML <tr> для определения строки таблицы;
  • Используйте элемент HTML <td> для определения ячейки данных таблицы;
  • Используйте элемент HTML <th> для определения заголовка таблицы;
  • Используйте элемент HTML <caption>, чтобы определить заголовок (подпись) таблицы;
  • Используйте свойство CSS borderдля определения границ таблицы;
  • Используйте свойство CSS border-collapse, чтобы свернуть границы ячеек;
  • Используйте свойство CSS padding для добавления отступов в ячейках;
  • Используйте свойство CSS text-align для выравнивания текста в ячейке
  • Используйте свойство CSS border-spacing для установки интервала (пространства) между ячейками;
  • Используйте атрибут colspan, чтобы сделать диапазон ячеек несколькими столбцами;
  • Используйте атрибут rowspan, чтобы сделать диапазон ячеек несколькими строками;
  • Используйте атрибут id, чтобы однозначно определить (идентифицировать) одну таблицу.

HTML теги для отображения таблицы

Тег Описание
<table> Определяет таблицу
<th> Определяет ячейку заголовка в таблице
<tr> Определяет строку в таблице
<td> Определяет ячейку в таблице
<caption> Определяет заголовок (описание) таблицы
<colgroup> Задает группу из одного или нескольких столбцов в таблице для форматирования
<col> Задает свойства столбца для каждого столбца в элементе <colgroup>
<thead> Группирует содержимое заголовка в таблице
<tbody> Группирует содержимое тела в таблице
<tfoot> Группирует содержимое нижнего колонтитула в таблице
