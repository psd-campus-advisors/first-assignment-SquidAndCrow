Изображения могут улучшить дизайн и внешний вид веб-страницы.

Примеры:

<img src="pulpitrock.jpg" alt="Вид на гору">
<img src="img_girl.jpg" alt="Девушка в пиджаке">
<img src="img_chania.jpg" alt="Цветы на улице">

Синтаксис HTML-изображений

В HTML изображения определяются тегом <img>. Тег <img> содержит только атрибуты и не имеет закрывающего тега. Тег <img> имеет два обязательных атрибута: src и alt.

Атрибут src

Атрибут src указывает URL (веб-адрес) изображения, т.е., где оно находится:

<img src="url">

Атрибут alt

Атрибут alt предоставляет альтернативный текст для изображения, если пользователь по какой-либо причине не может его просмотреть (например, из-за медленного соединения и работы Интернета, ошибки в атрибуте src или если пользователь использует программу чтения с экрана). Значение атрибута alt должно описывать изображение:

Пример 13.1:

<img src="img_chania.jpg" alt="Flowers in Chania">

Если браузер не может найти изображение, он отобразит значение атрибута alt:

<img src="wrongname.gif" alt="Flowers in Chania">
Посмотреть результат!

Т.е. в данном примере при отсутствии самого изображения отобразится надпись Flowers in Chania.

Примечание. Согласно спецификации HTML5 атрибут alt обязателен! Без него веб-страница не будет валидной.

Различия между HTML4.01 и HTML5

Атрибуты: align, border, hspace и vspace, которые ранее поддерживались в HTML4.01, не поддерживаются в HTML5. Хотя стоит заметить, что практически все браузеры правильно отобразят данные атрибуты, если они будут присутствовать в HTML-коде. Но при проверке веб-страницы на валидаторе, появится предупреждающее сообщение, что данные атрибуты устаревшие и рекомендуется использовать соответствующие свойства в каскадной таблице стилей - CSS.

Настройки CSS по умолчанию для тега <img>

img {
   display: inline-block;
}

Размер изображения - ширина и высота

Чтобы указать ширину и высоту изображения, используют атрибут style с указанием размеров изображения в пикселях.

<img src="img_girl.jpg" alt="Girl in a jacket" style="width:500px;height:600px;">

Кроме того, вы можете использовать атрибуты width и height в самом теге <img>. При этом размеры изображения достаточно указать только с помощью чисел (без указания единицы - пикселей px):

<img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">

Атрибуты width и height всегда определяют ширину и высоту изображения в пикселях.

Примечание. Всегда указывайте ширину и высоту изображения. Если ширина и высота не указаны, страница может мерцать во время загрузки изображения.

Ширина и высота или стиль?

Атрибуты width, height и style действительны в HTML5. Однако w3c предлагает использовать атрибут style. Он не позволяет таблицам стиля изменять размер изображений:

Пример 13.2:

<!DOCTYPE html>
<html>
<head>
<style>
/* Эта таблица стилей устанавливает ширину всех изображений на 100%: */
img {
width:100%;
}
</style>
</head>
<body>
<h3>Стилизация изображения</h3>
<p>Изображение ниже имеет атрибут width, установленный на 128 пикселей, но таблица стилей отменяет его и устанавливает ширину 100%.</p>
<img src="../img/html5.gif" alt="HTML5 Icon" width="128" height="128">

<p>В приведенном ниже изображении используется атрибут style, где ширина равна 128 пикселям, что переопределяет таблицу стилей</p>
<img src="../img/html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">
</body>
</html>
Посмотреть результат!

Т.е., если в каскадной таблице стилей указан размер изображения 100%, то все изображения на странице (или сайте) будут автоматически растягиваться на весь размер страницы, что не всегда бывает необходимо. Иногда нужно, чтобы размеры изображения были заданной фиксированной ширины и высоты. Для этого используется атрибут style для конкретного изображения на странице с указанием его размеров. Внутренний стиль элемента всегда имеет приоритет в отображении перед внешней таблицей стилей.

Изображения в другой папке

Если не указано, браузер ожидает найти изображение в той же папке, что и веб-страница. Однако обычно хранят изображения в подпапке. Вы должны указать имя папки в атрибуте src:

<img src="../img/html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

В данном примере изображение находится в папке img. Точки перед img с разделяющим слешем означают, что папка img находится в директории на уровень выше. Обычно папки для хранения изображений на сайте имеют название img или image (images).

Изображения на другом сервере

Некоторые веб-сайты хранят свои изображения на специальных серверах для хранения изображений. Фактически, вы можете получить доступ к изображениям с любого веб-адреса в мире, указав к нему путь:

Пример 13.3:

<img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools.com">
Посмотреть результат!

В данном примере отображается картинка, которая находится на сайте w3schools.com

Анимированные изображения

HTML позволяет использовать анимированные GIF-файлы:

Пример 13.4:

<img src="../img/programming.gif" alt="Computer Man" style="width:48px;height:48px;">
Посмотреть результат!

Вставка на веб-страницу анимированных GIF-файлов производится также, как и любых других изображений.

Изображение как ссылка

Чтобы использовать изображение в качестве ссылки, поместите тег <img> внутри тега <a>:

Пример:

<p>Изображение является ссылкой. Вы можете кликнуть на нём.</p>
<a href="https://google.com/" target="_blank">
<img src="../img/google_logo.png" alt="Поиск Google" style="width:120px;height:70px;border:0;">
</a>
Посмотреть результат!

В результате при клике на изображении оно будет работать как ссылка.

Примечание. Применяется border: 0; , чтобы IE9 (и ранее) не отображал границу вокруг изображения (когда изображение является ссылкой).

Обтекание изображения (свойство float)

Используйте свойство CSS float, чтобы изображение обтекало текстом справа или слева:

Пример 13.6:

<p><img src="../img/smiley.gif" alt="Smiley face" style="float:right;width:42px;height:42px>Изображение будет плавающим справа от текста.</p>
<p><img src="../img/smiley.gif" alt="Smiley face" style="float:left;width:42px;height:42px>Изображение будет плавающим слева от текста.</p>
Посмотреть результат!

В данном примере мало текста и само изображение небольшое. Чтобы увидеть, как работает свойство float, лучше посмотреть на страницу в уменьшенном окне.

Иногда говорят, что текст обтекает изображение слева или справа. Но при этом, если используется значение right (справа), то текст обтекает изображение слева, и наоборот, если используется значение left (слева), то текст обтекает изображение справа.

Совет. Чтобы узнать больше о CSS свойстве float, прочитайте учебник по CSS Float.

Карты изображений - image-map

Тег <map>

Используйте тег <map>, чтобы определить карту изображений. Карта изображения - это изображение с интерактивными областями. На данном изображении кликните на ноутбук, мобильный телефон или чашку кофе:

Пример 13.7:

<img src="../img/workplace.jpg" alt="Workplace" usemap="#workmap">
<map name="workmap">
<area shape="rect" coords="34,44,270,350" alt="Computer" href="computer.htm">
<area shape="rect" coords="290,172,333,250" alt="Phone" href="phone.htm">
<area shape="circle" coords="337,300,44" alt="Coffee" href="coffee.htm">
</map>
Посмотреть результат!

где:
map - карта;
name - имя;
usemap - пользовательская карта;
area - площадь, область;
shape - форма;
rect (от rectangle) - прямоугольник;
circle - круг;
coords (от coordinates) - координаты.

В данном примере при клике на каждом из объектов на картинке (на ноутбуке, мобильном телефоне и чашке кофе) мы будем переходить на новую страницу с описанием каждого из объектов. Если же мы попытаемся кликнуть где-нибудь в другом месте на картинке вне данных объектов, то ничего не произойдёт.

Атрибут name тега <map> связан с атрибутом usemap <img> и создает связь между изображением и картой. Тег <map> содержит несколько тегов <area>, которые определяют области, которые можно кликать на карте изображения.

Примечание. В HTML5, если атрибут id тега <map> указан, он должен иметь то же значение, что и атрибут name.

Настройки CSS по умолчанию для map

Большинство браузеров отобразит элемент <map> со следующими значениями по умолчанию:

map {
   display: inline;
}

Тег <area>

Тег <area> определяет область внутри карты изображения. Элемент <area> всегда вложен внутри тега <map>.
Примечание. Атрибут usemap в теге <img> связан с атрибутом имени элемента <map> и создает связь между изображением и картой.

В HTML5 имеется несколько новых атрибутов для тега <area>, а некоторые атрибуты HTML 4.01 больше не поддерживаются.

Настройки CSS по умолчанию для area

Большинство браузеров отобразит элемент <area> со следующими значениями по умолчанию:

area {
   display: none;
}

Атрибуты shape и coords

Атрибут shape определяет форму области. Атрибут shape используется вместе с атрибутом coords для указания размера, формы и размещения области.

Синтаксис площади формы:

<area shape="default|rect|circle|poly">
  • default - определяет всю область;
  • rect - определяет прямоугольную область;
  • circle - определяет круглую область;
  • poly - определяет многоугольную область.

Атрибут coords определяет координаты области на карте изображения.

Синтаксис определения координат формы:

<area coords="значения">
Координаты верхнего левого угла области равны 0,0.
Значение Описание
x1,y1,x2,y2 Задает координаты левого, верхнего, правого, нижнего угла прямоугольника (для shape = "rect")
x,y,radius Определяет координаты центра круга и радиуса (для shape = "circle")
x1,y1,x2,y2,..,xn,yn Задает координаты краев многоугольника. Если первая и последняя пары координат не совпадают, браузер добавит последнюю пару координат, чтобы закрыть многоугольник (для shape = "poly")

Фоновое изображение - background-image

Чтобы добавить фоновое изображение в элемент HTML, используйте свойство CSS background-image:

Пример 13.8:

Чтобы добавить фоновое изображение на веб-страницу, укажите свойство background-image элемента BODY:

<body style="background-image:url('../img/clouds.jpg')">
<h2>Background Image</h2>
<p>По умолчанию фоновое изображение будет повторяться, если оно меньше, чем элемент, где он указан, в данном случае элемент BODY.</p>
</body>
Посмотреть результат!

Пример 13.9:

Чтобы добавить фоновое изображение в параграфе, укажите свойство background-image на элементе P:

<body>
<p style="background-image:url('../img/clouds.jpg')">
Вы можете указать фоновые изображения для любого видимого элемента HTML. В этом примере фоновое изображение указано для параграфа. По умолчанию фоновое изображение будет повторяться в направлении (направлениях), где оно меньше элемента, в котором указано. Попробуйте изменить размер окна браузера, чтобы увидеть, как фоновое изображение ведёт себя.
</p>
</body>

Учитывайте, что стиль, прописанный непосредственно внутри элемента, имеет бо́льший приоритет, чем стиль, прописанный в отдельной таблице стилей!

Элемент <picture> (картинка)

HTML5 предоставил элемент <picture>, чтобы добавить бо́льшую гибкость при указании ресурсов изображения. Элемент <picture> содержит несколько элементов <source>, каждый из которых ссылается на разные источники изображений. Таким образом, браузер может выбрать изображение, которое наилучшим образом соответствует текущему виду страницы и/или устройству. Каждый элемент <source> имеет атрибуты, описывающие, когда их изображение является наиболее подходящим. Браузер будет использовать первый элемент <source> с соответствующими значениями атрибутов и игнорировать следующие элементы <source>.

Пример 13.10:

<picture>
<source media="(min-width: 650px)" srcset="../img/img_pink_flowers.jpg">
<source media="(min-width: 465px)" srcset="../img/img_white_flower.jpg">
<img src="../img/img_orange_flowers.jpg" alt="Flowers" style="width:auto;">
</picture>
Посмотреть результат!

В данном примере мы видим результат отображения изображения, если окно браузера (viewport) имеет минимум 650 пикселей, а другое изображение - меньше, но больше 465 пикселей. Изменяя окно браузера, мы видим, что изображение меняется.

Поэкспериментируйте с размерами окна браузера, изменяя его и посмотрите на результат.

Наиболее часто используется элемент <picture> для художественного оформления в гибких проектах. Вместо того, чтобы иметь одно масштабируемое или уменьшенное изображение на основе ширины окна просмотра, можно создать несколько изображений, чтобы более хорошо заполнить окно просмотра браузера.

Элемент <picture> содержит два разных тега: один или несколько тегов <source> и один тег <img>.

Элемент <source> имеет следующие атрибуты:

  • srcset (обязательный) - определяет URL-адрес изображения для показа;
  • media - принимает любой допустимый медиа-запрос, который обычно определяется в CSS;
  • size - определяет дескриптор одиночной ширины, один мультимедийный запрос с дескриптором ширины или список медиа-запросов с разделителями-запятыми с дескриптором ширины;
  • type - определяет тип MIME.

Атрибуты media, size и type не обязательные и используются при необходимости.

Браузер будет использовать значения атрибутов для загрузки наиболее подходящего изображения. Браузер будет использовать первый элемент <source> с соответствующим совпадением и игнорировать следующие теги <source>.

Элемент <img> требуется как последний дочерний тег блока объявления <picture>. Элемент <img> используется для обеспечения обратной совместимости для браузеров, которые не поддерживают элемент <picture>, или если ни один из тегов <source> не соответствует.

Элемент <picture> работает аналогично элементам <video> и <audio>. Вы настраиваете разные источники, и первым источником, который соответствует предпочтениям, является тот, который используется.

Тег <picture> является новым в HTML5.

Примечание. Всегда указывайте элемент <img> в качестве последнего дочернего элемента <picture> элемента. Элемент <img> используется браузерами, которые не поддерживают элемент <picture>, или если ни один из тегов <source> не был сопоставлен.

Краткое содержание главы

  • Используйте элемент HTML <img> для определения изображения;
  • Используйте атрибут HTML src для определения URL-адреса изображения;
  • Используйте атрибут HTML alt для определения альтернативного текста для изображения, если он не может быть отображен;
  • Используйте атрибуты HTML width (ширина) и height (высота), чтобы определить размер изображения;
  • Используйте свойства CSS width (ширина) и height (высота), чтобы определить размер изображения (альтернативно);
  • Используйте свойство CSS float, чтобы позволить обтекать изображения;
  • Используйте элемент HTML <map>, чтобы определить карту изображения;
  • Используйте элемент HTML <area> для определения областей, которые можно кликнуть на карте изображения.
  • Используйте атрибут usemap элемента HTML <img>, чтобы указать на карту изображения;
  • Используйте элемент HTML <picture> для отображения разных изображений для разных устройств;
Примечание. Загрузка изображений требует времени. Большие изображения могут замедлить вашу страницу. Чем больше объём и размеры изображения, тем дольше оно будет загружаться. Используйте изображения разумно.

HTML-теги изображения

Тег Описание
<img> Определяет изображение
<map> Определяет карту изображения
<area> Определяет область с кликами внутри карты изображения
<picture> Определяет контейнер для нескольких ресурсов изображения

width пиксели Задает ширину изображения.
