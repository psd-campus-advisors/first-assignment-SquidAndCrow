Приветствую вас на пятнадцатом уроке по HTML для начинающих, на котором вы узнаете о списках в HTML. Если вы только начинаете изучать HTML, то рекомендуется посмотреть также предыдущие мои уроки для начинающих (Плейлист на канале YouTube).


Списки на веб-странице выглядат также, как и в любом другом документе. Пример

Неупорядоченный список:

  • Вещь
  • Вещь
  • Вещь
  • Вещь

Упорядоченный список:

  1. Первая вещь
  2. Вторая вещь
  3. Третья вещь
  4. Четвертая вещь

Списки бывают разных видов.

Неупорядоченный (маркированный или ненумерованный) список в HTML

Неупорядоченный список начинается с тега <ul>. Каждый элемент списка начинается с тега <li>.

Элементы списка будут отмечены маркерами (по умолчанию в виде маленьких чёрных кругов):

Пример 15.1:

<ul>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ul>
Посмотреть результат!

В неупорядоченном (ненумерованном) списке можно выбирать вид маркера элемента списка.

CSS свойство list-style-type используется для определения стиля маркера элемента списка:

Значение Описание
disc Устанавливает маркер элемента списка в виде закрашенного круга (как и по умолчанию)
circle Устанавливает маркер элемента списка в виде круга
square Устанавливает маркер элемента списка в виде квадрата
none Элементы списка не будут отмечены (т.е. маркеры не отображаются)

Пример 15.2 – маркер disc:

<ul style="list-style-type:disc">
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ul>

Пример 15.3 – маркер circle:

<ul style="list-style-type:circle">
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ul>
Посмотреть результат!

Пример 15.4 – маркер square:

<ul style="list-style-type:square">
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ul>

Пример 15.5 – маркер none:

<ul style="list-style-type:none">
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ul>

Упорядоченный (пронумерованный) список в HTML

Упорядоченный список начинается с тега <ol>. Каждый элемент списка начинается с тега <li>.

Элементы списка будут помечены номерами по умолчанию:

Пример 15.6. Упорядоченный список:

<ol>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ol>
Посмотреть результат!

Упорядоченные списки могут иметь различные типы маркеров.

Атрибут type в упорядоченном списке

Атрибут type тега <ol> определяет тип маркера элемента списка:

Тип Описание
type="1" Элементы списка будут пронумерованы числами (по умолчанию)
type="A" Элементы списка будут пронумерованы заглавными буквами
type="a" Элементы списка будут пронумерованы строчными буквами
type="I" Элементы списка будут пронумерованы прописными римскими цифрами
type="i" Элементы списка будут пронумерованы строчными римскими цифрами

Элементы упорядоченного списка могут быть пронумерованы: числами (как и по умолчанию), заглавными буквами, строчными буквами, прописными римскими цифрами и римскими цифрами в нижнем регистре.

Тип "Числа":

Пример 15.7:

<ol type="1">
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ol>

HTML списки описания

HTML также поддерживает списки описания.

Список описания - это список терминов с описанием каждого термина.

Тег <dl> определяет список описания, тег <dt> определяет термин (название), а тег <dd> описывает каждый термин (название):

Пример 15.12:

<dl>
<dt>Coffee</dt>
<dd>- black hot drink</dd>
<dt>Milk</dt>
<dd>- white cold drink</dd>
</dl>
Посмотреть результат!

Вложенные HTML-списки

Список может быть вложенным (списки внутри списков):

Пример 15.13:

<ul>
<li>Coffee</li>
<li>Tea
<ul>
<li>Black tea</li>
<li>Green tea</li>
</ul>
</li>
<li>Milk</li>
</ul>
Посмотреть результат!
Примечание. Элементы списка могут содержать новый список и другие элементы HTML, такие как изображения и ссылки и т.д.

Подсчет контрольного списка

По умолчанию упорядоченный список начинает отсчет с 1. Если вы хотите начать подсчет с указанного числа, вы можете использовать атрибут start (т.е. начало):

Пример 15.14:

<ol start="50">
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ol>

Горизонтальный список с помощью CSS

HTML-списки можно стилизовать различными способами с помощью CSS.

Одним из популярных способов является создание списка по горизонтали для создания меню навигации:

Пример 15.15:

<!DOCTYPE html>
<html>
<head>
<style>
ul {
list-style-type: none;
margin: 0;
padding: 0;
overflow: hidden;
background-color: #333;
}

li {
float: left;
}

li a {
display: block;
color: white;
text-align: center;
padding: 16px;
text-decoration: none;
}

li a:hover {
background-color: #111;
}
</style>
</head>

<body>
<ul>
<li><a href="#home">Home</a></li>
<li><a href="#news">News</a></li>
<li><a href="#contact">Contact</a></li>
<li><a href="#about">About</a></li>
</ul>
</body>
</html>
Посмотреть результат!
Совет: вы можете узнать больше о CSS в нашем руководстве CSS.

Резюме

  • Используйте элемент HTML <ul> для определения неупорядоченного списка;
  • Используйте свойство CSS list-style-type для определения маркера элемента списка;
  • Используйте элемент HTML <ol> для определения упорядоченного списка;
  • Используйте атрибут HTML type для определения типа нумерации;
  • Используйте элемент HTML <li> для определения элемента списка;
  • Используйте элемент HTML <dl> для определения списка описания;
  • Используйте элемент HTML <dt> для определения термина описания;
  • Используйте элемент HTML <dd> для описания термина в списке описания;
  • Списки могут быть вложены в списки;
  • Элементы списка могут содержать другие элементы HTML;
  • Используйте свойство CSS float: left или display: inline для горизонтального отображения списка.

HTML теги списка

Тег Описание
<ul> Определяет неупорядоченный список
<ol> Определяет упорядоченный список
<li> Определяет элемент списка
<dl> Определяет список описания
<dt> Определяет термин в списке описания
<dd> Описывает термин в списке описания
