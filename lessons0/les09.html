Элемент Цитата

Quotation (анг.) - цитата, цитирование.

Цитаты на веб-страницах оформляются различными способами.

Элемент <blockquote> для цитат

Пример 9.1:

<p>Цитата бывшего чемпиона мира по боксу и мэра Киева Виталия Кличко:</p>
<blockquote cite="https://ru.wikipedia.org/wiki/Кличко,_Виталий_Владимирович">
А сегодня в завтрашний день не все могут смотреть. Вернее смотреть могут не только лишь все, мало кто может это делать...</blockquote>
Посмотреть результат!

В данном примере используется HTML-элемент <blockquote>, определяющий раздел, который цитируется из другого источника. Обычно для браузеров используются элементы отступов <blockquote>. Цитаты в HTML имеют свой стиль по умолчанию, который вы можете изменить либо непосредственно на самой странице в HTML-разметке, либо в каскадной таблице стилей (файле CSS).

Элемент <q> для коротких цитат

Элемент HTML <q> определяет короткую цитату. Браузеры обычно вставляют кавычки вокруг элемента <q>

Пример 9.2:

<p> Цитата Билла Гейтса: <q>Я всегда выберу ленивого человека делать трудную работу, потому что он найдет легкий путь её выполнения.</q></p>
Посмотреть результат!

Элемент <abbr> для аббревиатуры

Элемент HTML <abbr> определяет аббревиатуру или акроним. Маркировка аббревиатур может предоставить полезную информацию для браузеров, систем перевода и поисковых систем.

Пример 9.3:

<p><abbr title="HyperText Markup Language">HTML</abbr> - язык гипертекстовой разметки.</p>
<p><abbr>PHP</abbr> - скриптовый язык общего назначения, интенсивно применяемый для разработки веб-приложений.</p>
Посмотреть результат!

Как видно из примера, вместе с открывающим тегом <abbr> можно использовать атрибут title, в котором обычно прописывается расшифровка данной аббревиатуры. В таком случае в браузере слово-аббревиатура выделяется (подчеркивается) точечками (по умолчанию), а при наведении на него курсором появляется расшифровка аббревиатуры.

Элемент <address> для контактной информации

Элемент HTML <address> определяет контактную информацию (автора / владельца) документа или статьи. Элемент <address> обычно отображается курсивом. Большинство браузеров добавят разрыв строки до и после элемента.

Пример 9.4:

<address>
Автор: Студія "Веб-майстер".<br>
Відвідайте нас на:<br>
<a href="https://html5beginners.github.io" target="_blank">html5beginners.github.io</a><br>
а/с №88888888, м.Київ<br>
Україна<br>
</address>
Посмотреть результат!

Элемент <cite> для названия работы

Элемент HTML <cite> определяет название работы. Браузеры обычно отображают элементы <cite> курсивом.

Пример 9.5:

<p><cite>Умираючий гладіатор.</cite> Тарас Шевченко. Написана в 1856 році.</p>
Посмотреть результат!

Элемент <bdo> (Bi-Directional Override) для двунаправленного переопределения

Элемент HTML <bdo> определяет двунаправленное переопределение. Элемент <bdo> используется для переопределения текущего направления текста:

Пример 9.6:

<bdo dir="rtl">Этот текст будет написан справа налево </bdo>
Посмотреть результат!

Комментарии в HTML

Теги комментариев используются для вставки комментариев в исходный код HTML. Вы можете добавлять комментарии к своему HTML-коду, используя следующий синтаксис:

<!-- Напишите свой комментарий здесь -->

Обратите внимание, что в открывающем теге есть восклицательный знак (!), но его нет в закрывающем теге.

Примечание. Комментарии не отображаются браузером, но они могут помочь документировать исходный код HTML.

С комментариями вы можете размещать уведомления и напоминания в своем HTML:

Пример 9.7:

<h2>Это заголовок</h2>
<!-- Это комментарий -->
<p>Это параграф.</p>
<!-- Здесь можете добавить дополнительную информацию. -->
Посмотреть результат!

В браузере комментарии не отображаются, а служат только для служебных целей (т.е. для того, кто просматривает HTML-код).

Комментарии также хороши для отладки (дебагинга) HTML, потому что вы можете закомментировать HTML-строки кода, по одной, для поиска ошибок, если вдруг что-то пошло не так, например, стало отображаться не правильно после каких-то изменений в HTML-коде:

<!-- Не показывать это в данный момент
<img border="0" src="pic_mountain.jpg" alt="Эта картинка не будет отображаться внутри комментария">
-->

Условные комментарии

На некоторых веб-страницах вы можете встретить условные комментарии в HTML-коде:

<!--[if IE 9]> ... какой-то HTML-код здесь ... <![endif]-->

Условные комментарии определяют некоторые теги HTML, которые должны выполняться только браузером Internet Explorer. На более современных сайтах такие комментарии уже практически не используются.
