Урок 2. Редакторы HTML кода

В конце всё обязательно должно быть хорошо.
Если что-то плохо — значит, это ещё не конец…
(Пауло Коэльо)
  • С помощью каких программ пишется HTML-код
  • Редактор БЛОКНОТ
  • Редактор Notepad++
  • Редактор Brackets
  • Редактор Sublime Text 3
  • Редактор Atom

В этом уроке мы познакомимся с программами - редакторами HTML-кода, с помощью которых и пишется код веб-страниц. Сейчас существует довольно много различных редакторов HTML-кода. Еще каких-то 12-15 лет назад многих нынешних самых популярных HTML-редакторов даже не существовало. Лично я начинал писать свои первые веб-страницы с помощью стандартной программы на Windows - Блокнота. Но сейчас, конечно же, никто с помощью Блокнота HTML-код уже не пишет. Для этого существуют другие более удобные редакторы кода с подсветкой синтаксиса и различными функциями, которые облегчают написание кода. Первым моим HTML-редактором, который я начал использовать после стандартного Блокнота Windows стал Notepad++. Это свободный текстовый редактор с открытым исходным кодом для Windows с подсветкой синтаксиса большого количества языков программирования и разметки. Поддерживает открытие более 100 форматов. Базовая функциональность программы может быть расширена как за счёт плагинов, так и сторонних модулей, таких как компиляторы и препроцессоры. Именно с помощью Notepad++ можно начинать писать свои первые веб-страницы любым новичкам. Программа небольшая, довольно удобная, русифицированная, её можно скачать бесплатно с официального сайта. Есть портативный вариант программы, который не требует установки. Её достаточно распаковать в любую директорию на жестком диске вашего компьютера и пользоваться.

Второй редактор HTML-кода - Brackets - также бесплатный редактор с открытым кодом для веб-разработчиков. Brackets ориентирован на работу с HTML, CSS и JavaScript. Эти же технологии лежат в основе самого редактора, что обеспечивает его кроссплатформенность т.е. совместимость с операционными системами Mac, Windows и Linux. Большую функциональность этому редактору дают множество расширений (плагинов), добавляющих необходимые инструменты для работы над кодом.

Еще один замечательный редактор HTML-кода - это SublimeText3. Программа условно-бесплатная. Её можно скачать с официального сайта и пользоваться абсолютно бесплатно. Единственное неудобство в таком случае - это периодическое появление сообщения о необходимости купить программу. Также на официальном сайте SublimeText3 есть только англоязычная версия программы. Но на просторах Интернета легко можно найти и скачать пакеты с переводом на русский или украинский язык и, следуя рекомендациям, самостоятельно их установить в SublimeText. В целом этот HTML-редактор очень прост и понятен в использовании, но всё-таки рекомендуется именно для более опытных пользователей и верстальщиков сайтов, т.к. для расширения функциональности в SublimeText3 необходимо дополнительно доустановить некоторые дополнения - плагины, которых существует огромное множество. Но именно плагины и дают все те преимущества для этого редактора HTML-кода, которых нет в других программах.

Следующий редактор HTML-кода, и он же интегрированная среда разработки - это Atom. Атом предоставляет средства для кросс-платформенного редактирования кода, имеет интеллектуальную систему автодополнения ввода и многое другое. Есть множество дополнений и у этого редактора. Хотя для новичков он будет всё-таки довольно сложным, поэтому рекомендуется для более продвинутых пользователей.

Лично я пользуюсь редактором SublimeText3, который и рекомендую использовать всем новичкам для работы с HTML-кодом.

В Интернете можно, конечно же, найти и другие HTML-редакторы. Но все они, как правило, очень похожи на вышеперечисленные. Каждый может выбрать себе редактор по вкусу и степени сложности для освоения. Удачи!

В этом уроке рассказывается о наиболее популярных редакторах HTML-кода:

  • БЛОКНОТ (есть в ОС Windows);
  • Notepad++: notepad-plus-plus.org
  • Brackets: brackets.io
  • Sublime Text 3: sublimetext.com/3
  • Atom: atom.io

Другие редакторы и IDE (среда разработки):

  • Visual Studio Code: code.visualstudio.com
  • PHPStorm: jetbrains.com/phpstorm/
  • Dreamweaver: https://www.adobe.com/ua/products/dreamweaver.html
  • NetBeans: netbeans.org

Все эти редакторы можно скачать с официальных сайтов!
