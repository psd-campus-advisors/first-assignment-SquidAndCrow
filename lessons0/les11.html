Стилизация HTML с помощью CSS (каскадных таблиц стилей)

CSS - это каскадные таблицы стилей. CSS описывает, как элементы HTML должны отображаться на экране, на бумаге или на других носителях. CSS экономит много работы. Каскадные таблицы стилей могут контролировать расположение нескольких веб-страниц одновременно.

CSS можно добавить к элементам HTML тремя способами:

  • Встроенный (inline) - с помощью атрибута style в самих элементах HTML;
  • Внутренний (internal) - с помощью элемента <style> в разделе <head>;
  • Внешний - (external) с помощью внешнего файла CSS, подключенного к HTML-странице;

Наиболее распространенным способом добавления CSS является сохранение стилей в отдельных файлах CSS. Однако здесь мы будем использовать встроенный (inline) и внутренний (internal) стиль, потому что такие способы легче продемонстрировать и вам проще попробовать. Хотя на своих сайтах рекомендуется использовать подключение к HTML-странице именно внешних файлов CSS.

Вы можете узнать больше о CSS в учебнике по CSS на сайте w3schools.com

Встроенный CSS

Встроенный CSS используется для применения уникального стиля к одному элементу HTML. Встроенный CSS использует атрибут style для элемента HTML.

В этом примере цвет текста элемента <h2> отображается синим:

Пример 11.1:

<h2 style="color:blue;">Это заголовок синего цвета</h2>
Посмотреть результат!

Внутренний CSS

Внутренний CSS обычно используется для определения стиля одной HTML-страницы.

Внутренний CSS определен в разделе <head> HTML страницы в элементе <style>:

Пример 11.2:

<!DOCTYPE html>
<html>
<head>
<style>
body {background-color: powderblue;}
h1 {color: blue;}
p {color: red;}
</style>
</head>
<body>
<h1>Это заголовок</h1>
<p>Это параграф.</p>
</body>
</html>
Посмотреть результат!

В этом примере стиль задаётся элементам body (цвет фона страницы), заголовку h1 и параграфу p.

Внешний CSS

Внешняя таблица стилей используется для определения стиля многих HTML-страниц. С внешней таблицей стилей вы можете изменить внешний вид всего веб-сайта, изменив только один файл CSS! Чтобы использовать внешнюю таблицу стилей, добавьте в нее ссылку в разделе <head> на HTML-странице:

Пример:

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="styles.css">
</head>
<body>
<h2>Это заголовок</h2>
<p>Это параграф.</p>
</body>
</html>

Внешняя таблица стилей может быть написана в любом текстовом редакторе. Файл не должен содержать HTML-код и должен быть сохранен с расширением .css.

Вот как выглядит styles.css:

body {background-color: powderblue;}
h1 {color: blue;}
p {color: red;}

При подключении внешнего файла CSS результат отображения визульного оформления элементов на веб-странице будет таким же, как и при использовании встроенного и внутреннего стилей, но при этом общий вес (объём кода) страницы будет меньше. Кроме того, при изменении значений во внешней таблице стилей изменения происходят сразу на всех страницах, а не на одной, что значительно облегчает вёрстку многостраничного сайта, т.к. нет необходимости вносить изменения отдельно на каждой странице.

CSS шрифты. Color, font-family, font-size

Свойство CSS color определяет цвет текста, который будет использоваться.

Свойство CSS font-family определяет семейство шрифта, который будет использоваться.

Свойство CSS font-size определяет размер шрифта, который будет использоваться.

Пример 11.3:

<!DOCTYPE html>
<html>
<head>
<style>
 h1 {
  color: blue;
  font-family: verdana;
  font-size: 300%;
 }
 p {
  color: red;
  font-family: courier;
  font-size: 160%;
 }
</style>
</head>
<body>
<h1>Это заголовок</h1>
<p>Это параграф.</p>
</body>
</html>
Посмотреть результат!

CSS границы - Border

Свойство CSS border определяет границу вокруг элемента HTML:

Пример 11.4:

 p {
  border: 1px solid powderblue;
 }
Посмотреть результат!

CSS отступы - Padding

Свойство CSS Padding определяет дополнение (пробел, внутренний отступ) между текстом и границей (border):

Пример 11.5:

 p {
  border: 1px solid powderblue;
  padding: 30px;
 }
Посмотреть результат!

CSS отступы - Margin

Свойство CSS Margin определяет пробел (край, внешний отступ) за пределами границы (border):

Пример 11.6:

 p {
  border: 1px solid powderblue;
  margin: 50px;
 }
Посмотреть результат!

Атрибут идентификатора - id

Чтобы определить конкретный стиль для одного специального элемента, добавьте атрибут id к элементу:

Пример 11.7:

<p id="p01">Я отличаюсь от остальных. Я уникальный! </p>

затем определите стиль для элемента с конкретным идентификатором:

Пример 11.7:

#p01 {
  color: blue;
}
Посмотреть результат!

Т.е. параграф с идентификатором p01 будет иметь цвет: синий (blue).

Примечание. Идентификатор элемента должен быть уникальным на странице, поэтому селектор идентификаторов используется для выбора одного уникального элемента!

Атрибут класса - class

Чтобы определить стиль для особого типа элементов, добавьте атрибут класса к элементу:

<p class="error">Я отличаюсь</p>

затем определите стиль для элементов с определенным классом:

Пример 11.8:

 p.error {
  color: red;
}
Посмотреть результат!

Отличие между атрибутами идентификатора id и атрибутом класса class состоит в том, что идентификатор обычно используется для одного уникального элемента на странице, в то время, как атрибут класса используется для нескольких элементов на странице. Для обозначения идентификатора используется значок решетка (ещё называют диез или шарп), а для обозначения класса перед названием класса просто ставится точка.

Внешние ссылки

На внешние таблицы стилей можно ссылаться с полным URL-адресом или с указанием пути к текущей веб-странице.

В этом примере для ссылки на таблицу стилей используется полный URL-адрес:

Пример:

<link rel="stylesheet" href="https://www.w3schools.com/html/styles.css">

На некоторых сайтах данный способ используется для подключения внешних библиотек стилей.

В этом примере ссылка на таблицу стилей, которая находится в папке styles на текущем веб-сайте:

Пример:

<link rel="stylesheet" href="/styles/styles.css">

В этом примере ссылка на таблицу стилей, расположенную в той же папке, что и текущая страница:

Пример:

<link rel="stylesheet" href="styles.css">

Подробнее о путях файлов читайте в разделе «Пути файлов HTML»: https://www.w3schools.com/html/html_filepaths.asp

Краткое содержание главы

  • Используйте атрибут HTML style для встроенного стиля;
  • Используйте элемент HTML <style> для определения внутреннего CSS;
  • Используйте элемент HTML <link> для ссылки на внешний файл CSS;
  • Используйте элемент HTML <head> для хранения элементов <style> и <link>;
  • Используйте свойство CSS color для задания цвета текста;
  • Используйте свойство CSS font-family для текстовых шрифтов;
  • Используйте свойство CSS font-size для размера текста;
  • Используйте свойство CSS border для задания границ;
  • Используйте свойство CSS padding для пространства внутри элемента до границы;
  • Используйте свойство CSS margin для пространства за пределами границы элемента.
