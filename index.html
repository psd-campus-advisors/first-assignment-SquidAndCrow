HTML для начинающих. Уроки от "А" до "Я"

Viam supervadet vadens (лат. Дорогу осилит идущий)

Перед вами сайт, на котором можно абсолютно бесплатно пройти курс обучения языку HTML. Весь материал разделен на небольшие уроки, в каждом из которых будет максимально подробно изложен материал о каждой теме. Для новичков рекомендуется проходить уроки в том порядке, в котором они представлены. Но если вы уже ранее сталкивались с языком HTML (например, изучали его в школе) или даже пытались его учить самостоятельно посредством Интернета - смотрели видеоуроки или читали учебники - вам также пригодятся данные уроки. Здесь вы сможете найти именно те материалы, которые вам необходимы.

Уроки изложены максимально коротко, просто и понятно даже для новичков, которые ранее никогда не изучали язык HTML и не занимались написанием сайтов с помощью программ. Язык HTML не является языком программирования, на котором можно было бы написать какие-то программы. HTML является языком гипертекстовой разметки, с помощью которого пишется код, который считывается специальными программами - браузерами. И уже программы-браузеры отображают веб-страницы так, как это более привычно обычным пользователям Интернета.

На сайте представлены различные курсы - для новичков и для более продвинутых пользователей, которые уже ранее, возможно, изучали язык HTML или сталкивались с ним в процессе изучения веб-технологий и языков программирования.

В курсе для новичков вы сможете получить основные знания, необходимые для освоения и понимания написания кода HTML и сможете писать простейшие веб-страницы и статические сайты. Но если вы хотите научиться создавать полноценные динамические веб-сайты этих знаний, конечно же, будет недостаточно. Но в любом случае, фундамент уже будет заложен и базовые знания будут у вас в голове.

Чем эти уроки по изучению языка HTML для начинающих отличаются от других?

В сети Интернет есть множество сайтов, книг и видеоуроков, по которым можно изучать язык разметки HTML вместе с CSS. Но большинство из них написаны теми, кто просто банально пытается заработать денег (на книгах, сайтах или видео на Youtube). Часто это обычные любители либо студенты. Кроме того, и сайты, и книги, которые были написаны всего лишь 2 - 3 года назад, и видеоуроки на YouTube, очень быстро устаревают и очень редко поддерживаются своими авторами на протяжении длительного времени. Соответственно, учебные материалы об HTML / CSS также очень быстро теряют свою актуальность. То, что было стандартом еще каких-то 3-5 лет назад, сейчас уже просто становится не актуальным и не востребованным. Всё меняется, особенно в сфере web-технологий.

В своих уроках я вылаживаю материал, который является наиболее актуальным на момент создания этого сайта, при этом как-можно более подробно и доступно для понимания любому новичку, даже детям младшего школьного возраста, которые умеют более-менее нормально читать, пользоваться компьютером и Интернетом.

В разделе HTML / CSS для продвинутых пользователей находятся уроки для тех, кто уже имеет некоторые знания в HTML / CSS или уже прошел курс обучения для начинающих на этом сайте.

Также здесь используются материалы из первоисточников, официальных сайтов по технологиям HTML и CSS, таких как, например, сайт W3C (Консорциума Всемирной паутины) www.w3.org/, Живой Стандарт WHATWG, MDN.Mozilla и др.
